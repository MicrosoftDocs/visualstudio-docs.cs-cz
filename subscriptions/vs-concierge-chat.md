# <a name="concierge-chat"></a>Komunikace se specialistou

Specialistou služby poskytuje pomoc vyhledání informací o produktu a technickou dokumentaci stejně jako pomoc s využít výhody vašeho předplatného sady Visual Studio.

Služba je k dispozici 24 hodin denně, 7 a je v současné době nabízena v anglickém jazyce jenom (pro dotazy než účty, fakturace a předplatných).

## <a name="activation-steps"></a>Postup aktivace
Získání nápovědy prostřednictvím specialistou služby jsou velmi snadné.
1. Přihlaste se k [ https://my.visualstudio.com/benefits ](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).

2. Vyhledejte dlaždici specialistou služby v části Podpora stránce výhod a klikněte na **zahájit konverzaci**.
    > [!div class="mx-imgBorder"]
    > ![Dlaždice Concierge chatu](_img/vs-concierge-chat/vs-concierge-chat-tile.png)

3. Klikněte na tlačítko **pokračovat**.

4. Otevře se okno konverzace a zobrazí se vám čítače zobrazující svou pozici ve frontě.

5. Když se agent k dispozici, se spustí chat.
    > [!div class="mx-imgBorder"]
    > ![Concierge konverzace](_img/vs-concierge-chat/vs-concierge-chat-session.png)
