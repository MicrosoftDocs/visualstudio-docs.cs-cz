# <a name="visual-studio-cloud-subscriptions-billing-faq"></a>Nejčastější dotazy k fakturaci předplatných visual ateliéru
Nezapomeňte [porovnat výhody cloudového předplatného a ceny,](https://visualstudio.microsoft.com/vs/pricing/) abyste porozuměli výhodám každého předplatného sady Visual Studio, s porovnáním mezi předplatnými cloudu a standardního visual studia, podrobnostmi o výhodách pro předplatitele a dalšími.

## <a name="general-purchasing-questions"></a>Obecné otázky týkající se nákupu
### <a name="q-can-i-buy-visual-studio-cloud-subscriptions-using-a-purchase-order"></a>Otázka: Můžu si koupit předplatná cloudu Visual Studia pomocí nákupní objednávky?
Odpověď: Ne. Všechna předplatná Cloud u Visual Studia je nutné zakoupit pomocí předplatného Azure. (Představte si to jako fakturační účet Azure.)

### <a name="q-what-types-of-azure-subscriptions-can-be-used-to-buy-visual-studio-cloud-subscriptions"></a>Otázka: Jaké typy předplatných Azure se dá použít k nákupu cloudových předplatných Visual Studia?
A: Většinu předplatných Azure lze použít – podporujeme předplatná Azure připojená k vaší [podnikové smlouvě (EA),](https://azure.microsoft.com/pricing/enterprise-agreement/)předplatná Azure nastavená poskytovateli cloudových řešení (CSP), předplatná Azure nastavená prostřednictvím prodejců licencí Microsoft Open license a předplatná Azure s průběžnými platbami.

Některé typy předplatných Azure, včetně [bezplatné zkušební verze Azure](https://azure.microsoft.com/pricing/free-trial/) a předplatných zahrnutých jako výhody předplatitelů v předplatných Visual Studia, se nepoužívají.

### <a name="q-am-i-required-to-buy-other-azure-services"></a>Otázka: Musím si koupit další služby Azure?
A: Vůbec ne. Pokud si chcete koupit jenom cloudová předplatná Visual Studia přes Azure, můžete to udělat.

## <a name="enterprise-agreement-ea-customers"></a>Zákazníci s podnikovou smlouvou (EA)
### <a name="q-can-i-use-an-enterprise-agreement-to-buy-visual-studio-cloud-subscriptions"></a>Otázka: Můžu k nákupu předplatných cloudu sady Visual Studio použít smlouvu Enterprise?
A: Ano, můžete. Budete muset být vlastníkem nebo přispěvatelem předplatného Azure, které bylo vytvořeno pro vaši EA. Ujistěte se, že jste nákupy pro předplatná Visual Studio cloud přímo na webu Visual Studio Marketplace. Předplatná cloudu sady Visual Studio nelze zakoupit pomocí nákupní objednávky.

### <a name="q-how-can-i-tell-whether-i-have-the-necessary-privileges-to-buy-services-in-the-visual-studio-marketplace-through-my-organizations-enterprise-agreement"></a>Otázka: Jak poznám, jestli mám potřebná oprávnění k nákupu služeb na webu Visual Studio Marketplace prostřednictvím smlouvy Enterprise v organizaci?
A: Nejjednodušší přístup k určení, pokud máte práva oprávnění je kliknout na tlačítko **Koupit** pro službu nabízenou na webu Visual Studio Marketplace.
Musíte vybrat předplatné Azure (což je fakturační účet) z prezentovaného seznamu předplatných Azure, které jsou aktuálně propojené s vaším přihlášením.
Vzhledem k tomu, že název předplatného Azure výchozí typ fakturačního účtu ("Průběžně platby s průběžným platbou", "Enterprise Agreement", atd.), je často jasné, pokud předplatné Azure je součástí smlouvy Enterprise.

Dalším přístupem je pokus o návštěvu [portálu Azure Enterprise Portal](https://ea.azure.com).  Pokud se k němu dostanete úspěšně, pak už máte roli Enterprise Admin nebo Account Owner. Pouze vlastníci účtů můžou nastavit nové fakturační účty Azure ve smlouvě Enterprise. Pokud nemáte přístup k Portálu Azure Enterprise Portal, pak se v rámci vaší organizace zeptejte, kdo je váš podnikový správce, a požádejte tuto osobu, aby vás přidala jako vlastníka účtu v rámci Portálu Azure Enterprise Portal.  Pokud tuto osobu nemůžete najít, můžete [odeslat lístek podpory](https://support.microsoft.com/supportrequestform/cf791efa-485b-95a3-6fad-3daf9cd4027c) a požádat o kontaktní informace.  Pro lístek podpory potřebujete název organizace a registrační číslo smlouvy Enterprise.

### <a name="q-can-i-use-the-azure-monetary-commitment-funds-from-my-enterprise-agreement-to-buy-visual-studio-cloud-subscriptions"></a>Otázka: Můžu použít prostředky peněžního závazku Azure ze smlouvy Enterprise k nákupu cloudových předplatných Visual Studia?
A: Ne, tyto předplacené prostředky nejsou způsobilé pro nákup visual studio cloud předplatné. Když zvolíte předplatné Azure, které bylo vytvořeno pro vaše EA k nákupu cloudových předplatných Visual Studia, poplatky se zobrazí na vaší další faktuře "překročení". Obvykle se to děje měsíčně, ale kvůli historickým pravidlům pro některé zákazníky EA nemusí být faktura s nadbytkem vystavena několik měsíců. Pokud potřebujete vědět, jaké množství dalších nákupů (nákupů, které nejsou způsobilé pro finanční prostředky Azure Monetary Commitment), spustí fakturu za nadbytečnost, obraťte se na licenčního specialistu pro svou EA.

## <a name="how-charges-are-processed"></a>Způsob zpracování poplatků
### <a name="q-how-are-monthly-cloud-subscription-charges-processed"></a>Otázka: Jak **se** zpracovávají měsíční poplatky za předplatné cloudu?
A: Při prvním nákupu účtujeme poměrné množství, které pokrývá zbývající dny v aktuálním měsíci. Pokud například 15. dubna byl proveden nákup 10 měsíčních cloudových předplatných Visual Studio Professional, účtujeme 5 jednotek, protože 50 % měsíce zůstane (15 dní v 30denním měsíci).
Prvního května a poté každý měsíc, dokud jej nezrušíte, bude účtováno celých 10 jednotek.

Když později zvýšíte placené množství, zvýšíme také zvýšené jednotky, abychom pokryli zbývající dny v aktuálním měsíci. Pokud jste si tedy 1 další měsíční předplatné cloudu Visual Studio Professional koupili 10. května, vyfakturovali bychom zhruba 0,677 jednotek (v měsíci květen 31 dnů zbývá 21 dní).

### <a name="q-how-are-annual-cloud-subscription-charges-processed"></a>Otázka: Jak se zpracovávají **roční** poplatky za předplatné cloudu?
A: Při každém nákupu účtujeme celé množství zakoupené okamžitě. Poplatky nejsou rozloženy v průběhu roku a neexistuje žádné poměrné. Pokud si koupíte roční cloudové předplatné v různých obdobích v roce, budete mít předplatné obnovení v různých měsících. Neprovádíme všechny roční předplatná zákazníka v cloudu jako shodovat, jak je běžné při nákupu multilicenční smlouvy společnosti Microsoft.

### <a name="q-how-do-cancelations-work"></a>Otázka: Jak zrušení fungují?
A: Když zrušíte předplatné cloudu Sady Visual Studio, stornujete automatické obnovení. Předplatné pokračuje až do svého normálního data obnovení a pak jednoduše vyprší.
Po vypršení platnosti předplatitele sady Visual Studio již nelze použít Visual Studio nebo jiné výhody z předplatného.

U měsíčních cloudových předplatných se zrušení projeví první den následujícího měsíce. Pokud zrušíte jenom některá měsíční cloudová předplatná, nezapomeňte odebrat uživatele prvního příštího měsíce, abyste zajistili, že správným lidem budou i nadále přiřazena aktivní předplatná.

U ročních předplatných cloudových služeb se zrušení projeví první den měsíce následujícího po 12 měsících od původního nákupu nebo 12 měsíců od posledního ročního poplatku za obnovení. Pokud jste si například 3. ledna 2018 koupili roční cloudové předplatné Visual Studia Professional, zůstane aktivní až do 1. Pokud předplatné zrušíte kdykoli mezi 1. Neexistuje žádná sleva za zrušení části cesty přes rok předplatného s ročním icloudpředplatného.

### <a name="q-what-kind-of-volume-discounts-are-available-for-visual-studio-subscriptions"></a>Otázka: Jaký druh množstevních slev je k dispozici pro předplatná sady Visual Studio?
A: Získáte 5% slevu na 6th a všechny následné předplatné *v rámci každého typu* předplatného:

* Visual Studio Professional měsíčně
* Visual Studio Professional výroční
* Visual Studio Enterprise měsíčně
* Visual Studio Enterprise roční

Pokud si například koupíte 6 měsíčních předplatných Visual Studio Professional a měsíční předplatné 5 Visual Studio Enterprise, zaplatíte běžnou cenu za 5 Professional, získáte 5% slevu na 6. Odběry.

Sleva se také vztahuje pouze na poplatky v daném měsíčním fakturačním období. Takže pokud si koupíte 5 Visual Studio Professional roční předplatné v jednom měsíci, a pak si koupíte 5 více příští měsíc, budete platit běžnou cenu na všech 10 předplatných.

> [!NOTE]
> Microsoft už nenabízí roční předplatná Visual Studio Professional a roční předplatná Visual Studia Enterprise v předplatných cloudu. Stávající zákazníci se nezmění a možnost obnovit, zvýšit, snížit nebo zrušit jejich odběry. Noví zákazníci se [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/) doporučuje přejít na prozkoumat různé možnosti nákupu Visual Studio.

## <a name="other-questions"></a>Další otázky
### <a name="q-can-i-use-the-monthly-azure-devtest-individual-credit-as-a-visual-studio-subscriber-to-buy-more-visual-studio-cloud-subscriptions"></a>Otázka: Můžu použít měsíční kredit Azure DevTest jako předplatitel Visual Studia k zakoupení dalších předplatných cloudu Sady Visual Studio?
A: Ne, nemůžete použít [měsíční Azure DevTest individuální kredit](https://azure.microsoft.com/pricing/member-offers/credit-for-visual-studio-subscribers/) jako předplatitel Visual Studio platit za nákupy Visual Studio Marketplace. Všechny nákupy předplatného visual studia cloud se budou účtovat na vaši platební kartu.
Před nákupem budete muset [odebrat limit útraty](https://azure.microsoft.com/pricing/spending-limits/).

### <a name="q-whats-the-difference-between-annual-and-monthly-cloud-subscriptions"></a>Otázka: Jaký je rozdíl mezi ročním a měsíčním předplatným cloudu?
A: Měsíční cloudpředplatné patří Visual Studio plus použití Azure DevOps Services a TFS. Roční cloudová předplatná mají také toto, ale také zahrnují výhody předplatitelů, včetně použití Systému Windows a dalšího softwaru Microsoftu k instalaci a spuštění pro vývoj a testování, měsíční individuální kredit Azure DevTest, který se použije pro experimentování s Azure a vývoj a testování v cloudu, školení, podporu a mnoho dalšího.
[Porovnání výhod předplatného cloudu a cen](https://visualstudio.microsoft.com/vs/pricing/)

### <a name="q-do-i-get-new-versions-of-visual-studio-if-i-buy-a-visual-studio-cloud-subscription"></a>Otázka: Dostanu nové verze Visual Studia, když si koupím předplatné cloudu Sady Visual Studio?
A: Ano. Po vydání nových verzí si je můžete stáhnout a spustit. Navíc můžete i nadále spouštět předchozí verze.

### <a name="q-can-i-buy-visual-studio-cloud-subscriptions-from-my-software-reseller"></a>Otázka: Můžu si koupit předplatná cloudu Visual Studia od svého prodejce softwaru?
A: Ano, můžete, pokud váš prodejce se účastní programu Poskytovatele cloudových řešení (CSP). Prostě se jich zeptej.
