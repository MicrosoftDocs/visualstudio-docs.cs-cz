# <a name="introducing-visual-studio-subscriptions-with-github-enterprise"></a>Představujeme předplatná Visual Studia s GitHub Enterprise  

Zákazníci, kteří mají smlouvy Enterprise (EA) se společností Microsoft, si mohou zakoupit novou nabídku předplatného, která sdružuje standardní předplatná Sady Visual Studio a GitHub Enterprise. Je to snadný a ekonomický způsob, jak pro předplatitele Visual Studia získat GitHub Enterprise. 

- Inovujte ve velkém měřítku pomocí Visual Studia a GitHubu tím, že bezpečně přivedete open source kód, komunitu a osvědčené postupy do vašeho IDE a pracovních postupů.
- Používejte nejlepší nástroje a služby pro vývoj ve třídě pro vývojáře s nejkomplexnější sadou prostředků k vytváření, nasazování a správě skvělých aplikací. 
- Spolupracujte, spravujte žádosti o přijetí informací na GitHub U Enterprise a zkontrolujte zdrojový kód tam, kde byl napsán – v editoru. 

## <a name="whats-available"></a>Co je k dispozici? 

Pokud vaše organizace vstupuje do nové ea nebo obnovuje stávající, jsou k dispozici následující možnosti:

- Visual Studio Enterprise s GitHub Enterprise
- Visual Studio Professional s GitHub Enterprise

Pokud vaše organizace má EA, která ještě není kvůli obnovení, můžete zakoupit sku "Step-up" sku přidat GitHub Enterprise do stávajícího Visual Studio Enterprise nebo Visual Studio Professional předplatné. Poté, když se vaše smlouva obnoví, můžete si nabídku zakoupit.

Existující skum pro Visual Studio Enterprise, Visual Studio Professional a GitHub Enterprise bude také k dispozici samostatně. 

Chcete-li se dozvědět https://visualstudio.microsoft.com/subscriptions/visual-studio-githubvíce o výhodách a možnostech, navštivte . 

## <a name="getting-your-subscriptions"></a>Získání předplatných

Předplatná Visual Studio + GitHub Enterprise jsou přiřazena ve dvou částech:
1. Předplatné Visual Studia (Enterprise nebo Professional)
2. Předplatné GitHub Enterprise

Když vaše organizace zakoupí předplatná, přiřadí je správci. Předplatná Visual Studia a předplatná GitHub Enterprise se přidávají samostatně.  

Pokud se vám přidělí předplatné Visual Studia + GitHub, obdržíte e-mail s oznámením, že vám správce předplatného Sady Visual Studio přiřadil předplatné.  V tomto okamžiku budete moci vidět https://my.visualstudio.com/subscriptionsna .  

Vzhledem k tomu, že předplatná GitHub Enterprise jsou přiřazena samostatně, můžete obdržet oznámení od jiného správce v závislosti na tom, jak se vaše společnost rozhodla uspořádat svá předplatná.  Může být také zpoždění mezi přiřazení předplatného Visual Studio a předplatné GitHub Enterprise, takže se nemusíte obávat, pokud nejsou oba okamžitě k dispozici.  Proces přiřazení předplatného GitHub je složitější a může trvat několik pracovních dnů.

## <a name="frequently-asked-questions"></a>Nejčastější dotazy

### <a name="q--where-is-my-github-subscription"></a>Otázka: Kde je moje předplatné GitHub?
A: Proces nastavení předplatného GitHub může chvíli trvat.  Pokud jste čekali a nic jste neslyšeli, možná budete chtít kontaktovat správce GitHubu.  Je možné, že na tom správce stále pracuje nebo se rozhodl nepřiřazovat vám předplatné GitHubu. 

### <a name="q--how-do-i-reach-my-github-administrator"></a>Otázka: Jak se dostanu ke správci GitHubu?
A: Pokud používáte GitHub Enterprise Server, můžete kliknout na odkaz **podpory** viditelné v dolní části klienta, které mají být spojeny se správcem.

### <a name="q-do-i-get-the-full-visual-studio-subscription-professional-or-enterprise-or-do-i-just-get-the-ide-with-this-offering"></a>Otázka: Mám získat úplné předplatné Visual Studia (Professional nebo Enterprise) nebo mám jen získat ide s touto nabídkou?
A: Spolu s github enterprise předplatné, obdržíte úplné předplatné Visual Studio.  Získáte všechny výhody, jako je stahování softwaru, cloudové služby, technická podpora a výhody partnerů zahrnuté v těchto předplatných, stejně jako v případě, že předplatné bylo zakoupeno samostatně.
