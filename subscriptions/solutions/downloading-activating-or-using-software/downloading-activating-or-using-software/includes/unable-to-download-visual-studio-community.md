## <a name="im-unable-to-download-or-activate-visual-studio-community"></a>Nedaří se mi stáhnout nebo aktivovat Visual Studio Community
Visual Studio Community je bezplatná, odlehčená verze integrovaného vývojového prostředí (IDE) a k jejímu používání nepotřebujete placené předplatné sady Visual Studio. Můžete [si ho stáhnout přímo](https://visualstudio.microsoft.com/vs/community/) z webu sady Visual Studio.
