# <a name="managing-subscriptions"></a>Správa předplatných

Visual Studio [portál pro předplatitele](https://my.visualstudio.com) dává možnost spravovat všechny odběry a programu ve skupinách na jednom místě. Můžete provést změny vašeho profilu, všechny výhody, které jsou součástí předplatných a přepnout mezi předplatnými rychle a snadno používat.

## <a name="signing-in-to-the-portal"></a>Přihlášení k portálu

Vítá vás předplatné Visual Studio!  Vaše předplatné vám poskytne veškerý software a výhody, které budete potřebovat aby se váš kód ubíral, včetně softwaru, měsíční kredity Azure, nástrojů pro spolupráci, vzdělávání Zkrátka, podpory a další – nejnovější a nejlepší od Microsoftu. Chcete-li použít portál, budete potřebovat k přihlášení.

### <a name="first-time-access"></a>Přístup poprvé

Postup používání vašeho předplatného pro první závisí na způsobu získání předplatného.

#### <a name="microsoft-store"></a>Microsoft Store

Pokud jste zakoupili předplatné sady Visual Studio přes Microsoft Store, postup aktivace můžete zahájit platby **potvrzovací stránku** nebo **potvrzovací E-mail.**

1. Na stránce"potvrzení" nebo "Potvrzení e-mailu", klikněte na **kliknutím sem se můžete přihlásit**. Budete přesměrováni na https://my.visualstudio.com/subscriptions/activate.
2. Zadejte e-mailovou adresu, které použijete k přihlášení k portálu pro předplatitele a klikněte na tlačítko **pokračovat**.
3. V závislosti na e-mailovou adresu může přesměrováni na přihlašovací stránku Microsoft Account nebo přihlašovací stránku vaší společnosti nebo školy.
4. Zadejte svoje heslo, nebo dokončete požadavky na přihlášení.
5. Zadejte kód Product Key uvedené v potvrzovací stránku nebo Uvítacího e-mailu a klikněte na tlačítko **aktivovat**.

#### <a name="workschool-accounts"></a>Pracovních nebo školních účtů

Pokud vaše předplatné poskytla vaše pracovní/školní, jste přijali Uvítacího e-mailu.

1. Přejděte [portál pro předplatitele](https://my.visualstudio.com/)
2. Zadejte e-mailovou adresu, které bylo přidruženo nové předplatné sady Visual Studio. Tato adresa je uvedena také v odběrateli uvítací e-mail. Složky Nevyžádaná pošta Zkontrolujte prosím, jestli nebyl přijat Uvítacího e-mailu.
3. Klikněte na tlačítko **pokračovat**
4. Budete přesměrováni na podnikové přihlašovací stránku.
5. Zadejte své heslo.
6. Klikněte na tlačítko **přihlášení**.

#### <a name="visual-studio-marketplace"></a>Visual Studio Marketplace

Pokud jste koupili měsíční nebo roční cloudové předplatné z webu Visual Studio Marketplace, navštivte [portál pro předplatitele](https://my.visualstudio.com/) a přihlaste se pomocí účtu Microsoft (MSA), které jste zadali jste si koupili předplatné.

#### <a name="using-a-workschool-account-with-a-microsoft-account-msa"></a>Pomocí pracovní nebo školní účet pomocí účtu Microsoft (MSA)

Pokud má vaše organizace službu Azure Active Directory, ale nemáte účet Azure Active Directory, možná budete muset využít **osobní** možnosti během procesu přihlašování.

1. Přejděte prosím [portál pro předplatitele](https://my.visualstudio.com)
2. Zadejte e-mailovou adresu, které bylo přidruženo nové předplatné sady Visual Studio. Tato adresa je uvedena také v odběrateli **uvítacím dopise**. (Pokud jste nedostali Uvítacího e-mailu, zkontrolujte složky s nevyžádanou poštu.)
3. Klikněte na tlačítko **pokračovat**
4. Budete přesměrováni na **rozhodnutí stránky**.  Vyberte **pracovní nebo školní účet** Pokud vaše předplatné je přiřazena k **práce nebo do školy** účet spojený s tenantem Azure Active Directory (AAD). Vyberte **osobní** Pokud předplatné je přidružené k **podnikové** e-mailová adresa, ale zároveň bylo převedeno na **osobní** účtu Microsoft (MSA)
5. Zadejte svoje heslo
6. Klikněte na tlačítko **přihlášení**

- Může být vypršení platnosti. Jenom aktivní odběry se zobrazují v portálu pro předplatitele.
- Může být přidružena jinou e-mailovou adresu. Například pokud se přihlášení k předplatnému sady Visual Studio Professional s vaší pracovní nebo školní e-mailem a máte Visual Studio Dev Essentials členství přidružené Account Microsoft, uvidíte pouze předplatné odpovídající e-mailovou adresu použít pro přihlášení k portálu pro předplatitele. Pokud chcete zobrazit jiné předplatné, je potřeba Odhlásit se a přihlaste se pomocí e-mailovou adresu.
- Pokud vaše společnost kupuje předplatná prostřednictvím multilicence, váš správce nemusí mít přiřazené předplatné, nebo může změně přiřazení předplatného na jiného uživatele.

## <a name="renewing-my-subscriptions"></a>Prodlužuje se platnost mé odběry

**Standardní předplatná**

- Microsoft Store: Pokud jste si koupili předplatné z [Microsoft Store](http://www.microsoft.com/store), můžete ho obnovíte existuje také.
- Volume License: Pokud vaše společnost kupuje předplatná prostřednictvím multilicence, byste kontaktovat správce prodejce k obnovení vašich předplatných.

**Cloudová předplatná:** cloudovými předplatnými není potřeba obnovit. Tak dlouho, dokud se placené předplatné, zůstane aktivní.

**Odběry, které získali prostřednictvím programů:** Pokud se zobrazí vaše předplatné prostřednictvím programu, vaše předplatné se váže k vaší účasti v programu. Kontaktujte prosím program od svých kontaktů pro další informace.

- [Představte si](https://imagine.microsoft.com/about)
- [BizSpark](https://bizspark.microsoft.com/About/Offers)
- [Microsoft Partner Network](https://partner.microsoft.com/en-us)
- [Microsoft Certified Trainer](https://www.microsoft.com/en-us/learning/mct-certification.aspx)

**Visual Studio Dev Essentials:** Dev Essentials je bezplatný program a členství ve skupinách, nevyprší platnost.

## <a name="managing-my-profile"></a>Správa Můj profil

Na portálu předplatných sady umožňuje vytvářet a spravovat profil, vám usnadní správu předplatných. Vaše informace o identitě můžete měnit a vaše předplatná můžete přidružit jiné identity přihlášení z existujících účtů Azure nebo Azure DevOps organizace, které používají alternativní identity.

### <a name="changing-profile-information"></a>Změna informací o profilu

Můžete snadno změnit kontaktní informace a předvolby ve vašem profilu.

1. Klikněte na své jméno v pravém horním rohu portálu pro předplatitele.
2. Klikněte na tlačítko **upravit profil** tlačítko.
3. Na **o** kartu, můžete změnit název, e-mailu, země/oblast, obrázku a kontaktní předvolby profilu. Na **Předvolby** kartu, můžete vybrat preferovaný jazyk, data a času, časové pásmo a motiv uživatelského rozhraní.
4. Když jste dokončili změny, klikněte na tlačítko **Uložit**.
5. Klikněte na tlačítko v prohlížeči **zpět** tlačítko pro návrat stránku výhody.

   > [!div class="mx-imgBorder"]
   > ![Úprava informací profilu](_img/manage-vs-subscriptions/edit-profile-resized.png)

### <a name="linking-my-subscription-to-existing-azure-devops-services-or-azure-subscriptions"></a>Propojení předplatného Azure nebo předplatné na stávající služby Azure DevOps

Pokud se přihlásit k předplatnému pomocí osobního účtu Microsoft (MSA) a s předplatným, které zahrnuje Azure DevOps služby nebo Microsoft Azure pomocí svého pracovního nebo školního účtu, které spravuje Azure, můžete propojit dva odběry. To se nazývá "Alternativní účet" a umožňuje nadále používat MSA k přihlášení ke svému předplatnému přitom má přístup ke stávajícím prostředkům Azure, Azure DevOps služby nebo integrovaného vývojového prostředí, které jsou spojeny s váš školní nebo pracovní účet.

I když oba účty používají stejnou e-mailovou adresu, musíte přidat pracovní nebo školní účet jako alternativu k použití výhody prostřednictvím obou účtů.

Chcete-li přidat alternativní účet:

1. Přihlaste se k [portál pro předplatitele](https://my.visualstudio.com?wt.mc_id=o~msft~docs) pomocí svého účtu Microsoft (MSA).
2. Klikněte na **předplatná** kartu.
3. V části **související odkazy**, zvolte **přidat alternativní účet**.
4. Zadejte svou pracovní nebo školní účet e-mailovou adresu a klikněte na tlačítko **přidat**.

   > [!div class="mx-imgBorder"]
   > ![Alternativní účet](_img/manage-vs-subscriptions/alternate-account-resized.png)

Po přidání alternativního účtu, můžete aktualizovat nebo odebrat kdykoli kliknutím **předplatná** kartu a vyberete příslušnou možnost v **související odkazy** oddílu.
