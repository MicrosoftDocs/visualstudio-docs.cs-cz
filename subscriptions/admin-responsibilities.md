# <a name="overview-of-administrator-responsibilities"></a>Přehled odpovědností správce
Jako správce můžete spravovat předplatná pro vaši organizaci.  Role správce také nese odpovědnost, aby bylo zajištěno, že předplatná jsou spravována v souladu s licenčními podmínkami. Tento článek popisuje odpovědnosti, výhody a omezení role správce.

## <a name="roles--responsibilities"></a>Role & odpovědností
Správce sady Visual Studio má čtyři klíčové povinnosti:

1. **Seznamte se s výhodami a omezeními předplatných sady Visual Studio.** Správné pochopení výhod vám může umožnit snížit náklady na hardware pomocí cloudových služeb a snížit softwarové náklady pomocí licencí pro jednotlivé uživatele pro předprodukční prostředí. 
2. **Přiřaďte předplatná sady Visual Studio konkrétním, pojmenovaným osobám a podporu využití.** Vaše smlouva vyžaduje, aby byla předplatná sady Visual Studio přiřazena konkrétním, pojmenovaným osobám. Sledujte s přiřazenými jednotlivci a ujistěte se, že mají přístup a plně využijte výhod zahrnutých v jejich předplatném sady Visual Studio.
3. **Přesně inventarizovat vaše předprodukční prostředí.** To je nezbytné k zajištění toho, aby všichni uživatelé, kteří pracují se softwarem s licencí sady Visual Studio, měli odpovídající licenci s vlastním předplatným sady Visual Studio. 
4. **Sledujte změny přiřazení uživatelů a získejte další licence podle plánu.** Multilicenční smlouvy (VL) společnosti Microsoft a mpsa poskytují flexibilitu při používání a přiřazování předplatných sady Visual Studio. Na oplátku se očekává, že budete sledovat změny v používání softwaru a přiřazení uživatelů a zpracovávat objednávky pro další licence podle plánu uvedeného ve smlouvě.  Chcete-li pomoci s touto odpovědností, je [maximální využití sestavy](maximum-usage.md) můžete spustit na portálu pro správu předplatného sady Visual Studio. 

## <a name="benefits-and-limitations"></a>Výhody a omezení
Předplatná sady Visual Studio umožňují členům vývojového týmu instalovat a používat software k navrhování, vývoji, testování, vyhodnocování a předvádění dalšího softwaru. Software Visual Studio Subscriptions není licencován pro produkční prostředí.
