# <a name="aspnet-core-web-app-deployment"></a>ASP.NET nasazení základní webové aplikace

Důležitým krokem při vývoji webových aplikací je publikování (nasazení). Můžete publikovat do vzdáleného prostředí, takže ostatní mohou používat aplikaci nebo pro účely testování. Visual Studio pro Mac nabízí dvě možnosti publikování aplikace. Publikování do služby Azure App Service umožňuje publikovat webovou aplikaci do služby Azure App Service. Publikování do složky umožňuje publikovat aplikaci do složky.

## <a name="publish-to-azure-app-service"></a>Publikování do Azure App Service

Pomocí Visual Studia pro Mac můžete publikovat webovou aplikaci přímo do služby Azure App Service. Pokud chcete publikovat ASP.NET webovou aplikaci Core do služby Azure App Service, přečtěte si článek [publikování na azure app service.](publish-app-svc.md)

## <a name="publish-to-folder"></a>Publikování do složky

Můžete také publikovat webové aplikace ASP.NET Core do složky. Další informace o publikování webové aplikace do složky naleznete v [odkazech Publikovat do složky](publish-folder.md).
