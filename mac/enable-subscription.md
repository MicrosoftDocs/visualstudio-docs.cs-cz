# <a name="enable-subscription"></a>Povolení předplatného

Visual Studio pro Mac nabízí tři různé úrovně předplatného, ze kterých si můžete vybrat, všechny přístupné z jednoho stažení:

* Visual Studio Enterprise pro Mac
* Visual Studio Professional pro Mac
* Visual Studio Společenství pro Mac

Pokud chcete začít používat předplatné, přihlaste se k Visual Studiu pro Mac podle pokynů v průvodci [Přihlášení k Visual Studiu pro Mac.](signing-in.md) Přihlášení aktivuje vaše předplatné, které se zobrazí v dialogovém okně Účty:

![Dialogové okno Zobrazit uživatelskou licenci](media/user-accounts-login.png)

## <a name="update-expired-licenses"></a>Aktualizace licencí, jejichž platnost vypršela

Pokud platnost předplatného vypršela, máte dvě možnosti:

1. Obnovte předplatné. Předplatné můžete obnovit procházením [https://visualstudio.microsoft.com/vs/pricing/](https://visualstudio.microsoft.com/vs/pricing/)aplikace .

2. Pokud máte jiný účet s aktivním předplatným, můžete ho přidat do Visual Studia pro Mac, jak je podrobně popsáno v článku [Přidání více uživatelských účtů.](signing-in.md) Visual Studio for Mac rozpozná nejlepší dostupnou licenci ze všech účtů, které jste přidali.

## <a name="product-key-usage"></a>Použití kódu Product Key

V současné době neexistuje žádný způsob, jak využít kód Product Key k povolení profesionálních nebo podnikových nároků pro Visual Studio for Mac. Chcete-li použít Visual Studio Enterprise pro Mac nebo Visual Studio Professional pro Mac, musíte mít příslušné [předplatné](https://visualstudio.microsoft.com/subscriptions/) a být [přihlášeni](signing-in.md) k ide.

## <a name="offline-activation"></a>Offline aktivace

> [!NOTE]
> Offline aktivace není pro Visual Studio for Mac dostupná.
> Pokud chcete aktivovat předplatné Visual Studia 2019 pro Mac Professional nebo Enterprise, musíte být při přihlášení připojeni k internetu.

Pokud jste zakoupili Visual Studio Enterprise 2019 for Mac pro počítače bez připojení k internetu, můžete pokračovat v používání sady Visual Studio for Mac bez aktivace, ale funkce Xamarin Inspector, Xamarin Profiler a embedded sestavení nebudou K dispozici.
