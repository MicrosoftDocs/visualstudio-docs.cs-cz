# <a name="visual-studio-for-mac-tools-for-unity"></a>Visual Studio pro Mac Tools for Unity

![Stylizovaná image Unity a Visual Studio pro Mac](media/vsmac-tools-unity-image1.png)

Visual Studio for Mac Tools for Unity je bezplatné rozšíření sady Visual Studio, které promění sadu Visual Studio for Mac ve výkonný nástroj pro vývoj her a aplikací pro různé platformy pomocí platformy Unity.

Integrace Unity je součástí Visual Studio pro Mac a od Unity 2018,1 je Visual Studio pro Mac výchozím jazykem C# IDE pro projekty Unity.

> [!NOTE]
> Tyto informace jsou pro Visual Studio pro Mac IDE. Informace o Visual Studio Code najdete v [této příručce](https://code.visualstudio.com/docs/other/unity) .

## <a name="overview"></a>Přehled

Tady jsou některé klíčové funkce Visual Studio pro Mac nástrojů pro Unity:

### <a name="compatible-with-visual-studio-for-mac-community-edition"></a>Kompatibilní s Visual Studio pro Mac Community Edition

[Edice Visual Studio pro Mac Community](https://visualstudio.microsoft.com/) jsou zdarma a společně s ní se instalují s instalací Unity od verze Unity 2018,1. Další informace najdete v [dokumentaci k instalaci](setup-vsmac-tools-unity.md) nástrojů Visual Studio pro Mac Tools for Unity.

### <a name="intellisense-for-unity-messages"></a>IntelliSense pro zprávy Unity

Technologie IntelliSense umožňuje rychle a snadno [implementovat zprávy Unity](using-vsmac-tools-unity.md#intellisense-for-unity-messages) jako `OnCollisionEnter` včetně jejich parametrů.

### <a name="superior-debugging"></a>Vynikající ladění

Visual Studio pro Mac Tools for Unity podporují všechny funkce [ladění](using-vsmac-tools-unity.md#unity-debugging) , které očekáváte od sady Visual Studio:

* Nastavte zarážky, včetně podmíněných zarážek.
* Vyhodnotit složité výrazy v okno Kukátko.
* Zkontrolujte a upravte hodnotu proměnných a argumentů.
* Přechod k podrobnostem o komplexních objektech a datových strukturách.

### <a name="powerful-refactoring-and-context-actions"></a>Výkonné refaktoring a kontextové akce

Pomocí rychlých nabídek a klávesových zkratek pro [přejmenování, refaktoring a kontextovým akcím](refactoring.md)zapište více použitelného kódu.

### <a name="roslyn-diagnostics-and-quick-fixes-for-unity"></a>Diagnostika Roslyn a rychlé opravy pro Unity

Pište lepší kód, který zachycuje osvědčené postupy s využitím hloubkového porozumění projektů Unity Visual Studio pro Mac. 

![Porovnání řetězců refaktoringu pro Mac s CompareTag](media/using-vsmac-tools-unity-image9.png)

### <a name="browse-and-add-new-files"></a>Procházet a přidat nové soubory

Procházejte projekty Unity a [přidejte složky, skripty nebo shadery](using-vsmac-tools-unity.md#adding-new-unity-files-and-folders), a to vše v rámci Visual Studio pro Mac integrované vývojové prostředí (IDE).

### <a name="use-familiar-key-bindings"></a>Použití známých klíčových vazeb

Zvyšte produktivitu pomocí klíčových vazeb, které znáte. Visual Studio pro Mac poskytuje známé [klíčové vazby](customizing-the-ide.md) pro spoustu oblíbených systémů, jako je například Visual Studio ve Windows, reostřejšíer, Visual Studio Code a Xcode.

### <a name="customize-the-visual-theme"></a>Přizpůsobení vizuálního motivu

Dejte svým očí svůj pohled na zbytek s zahrnutým [tmavým motivem](customizing-the-ide.md).
