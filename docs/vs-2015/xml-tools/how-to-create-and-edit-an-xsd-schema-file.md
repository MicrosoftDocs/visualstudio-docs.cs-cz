# <a name="how-to-create-and-edit-an-xsd-schema-file"></a>Postupy: vytvoření a úprava souboru schématu XSD
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

  
Toto téma popisuje, jak vytvořit nový soubor schématu XML (XSD) a pak přidejte obsah do souboru XSD.  
  
### <a name="to-create-a-new-xml-schema-file"></a>Chcete-li vytvořit nový soubor schématu XML  
  
1.  V sadě Visual Studio, otevřete **souboru** nabídky a vybereme **nový** a potom **souboru**. Nebo použijte klávesovou zkratku CTRL + N.  
  
2.  V **nový soubor** dialogu **schématu XML** a klikněte na tlačítko **otevřít**.  
  
     Je vytvořen nový soubor. Počáteční zobrazení a prázdné okno Průzkumníka schémat XML se zobrazí v sadě Visual Studio.  
  
### <a name="to-edit-a-file"></a>Chcete-li upravit soubor  
  
1.  Klikněte na tlačítko **pomocí editoru XML k zobrazení a úpravě základního souboru schématu XML** odkaz na zobrazení spuštění.  
  
     XML Editor se zobrazí se nový soubor otevřít.  
  
2.  Zkopírujte ukázkový kód XML schéma z [nákupní pořadí schématu](../xml-tools/sample-xsd-file-simple-schema.md) a vložte ji nahradit kód, který byl přidán do nového souboru XSD ve výchozím nastavení.  
  
     Průzkumníka schémat XML je aktualizován nové sadě schémat.
