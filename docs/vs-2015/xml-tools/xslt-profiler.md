# <a name="xslt-profiler"></a>Profiler XSLT
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Profiler XSLT je nástroj profileru pro analýzu výkonu, který pomáhá vyvíjet a ladit dokumenty XSLT. Profiler XSLT umožňuje vývojářům měřit a hodnotit problémy související s výkonem v kódu XSLT vytvořením podrobných sestav výkonu XSLT. Profiler XSLT obsahuje užitečné rady pro optimalizace stylů XSL a XSLT, které jsou nezbytné pro aplikace založené na jazyce XSLT, které vyžadují maximální výkon.

 Profiler XSLT je součástí sady Visual Studio 2010 a je k dispozici v nabídce **XML** sady Visual Studio.

 ![Profiler XSLT](../xml-tools/media/xsltprofilermenu.gif "XSLTProfilerMenu")

 Profiler XSLT byl poprvé zaveden jako doplněk pro Visual Studio 2008. Další informace najdete v [doplňku XSLT profileru](https://marketplace.visualstudio.com/items?itemName=SinanUssakli-MSFT.XSLTProfilerAddin).

> [!NOTE]
> Některé obrazovky se mohou v aplikaci Visual Studio 2010 zobrazit jinak než v těchto dokumentech, protože některé vizuální prvky byly od sady Visual Studio 2008 změněny.
