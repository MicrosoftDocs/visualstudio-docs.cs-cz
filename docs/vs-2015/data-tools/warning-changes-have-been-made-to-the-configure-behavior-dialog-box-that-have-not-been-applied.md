# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Upozornění: V dialogovém okně konfigurace chování došlo ke změnám, které nebyly použity.
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění: V dialogovém okně Konfigurovat chování byly provedeny změny, které nebyly aplikovány. Chcete změny použít?

 Dialogové okno **Konfigurovat chování** umožňuje nakonfigurovat chování funkce INSERT, Update a DELETE pro všechny dostupné třídy. Tato zpráva se zobrazí, když vyberete novou kombinaci **třídy** a **chování** a předchozí změna ještě nebyla použita.

### <a name="to-apply-the-change-and-continue"></a>Použití změny a pokračování

- Kliknutím na **Ano** aplikujete změnu na vybranou **třídu** a **chování**.

### <a name="to-cancel-the-previous-change-and-continue"></a>Zrušení předchozí změny a pokračování

- Kliknutím na **ne** zrušíte změnu vybrané **třídy** a **chování**.
