# <a name="localizeddescription-element-vsix-language-pack-schema"></a>Element LocalizedDescription (VSIX Language Pack Schema)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Požadováno. Poskytuje lokalizovaný popis rozšíření.

```  
<LocalizedDescription>Localized description of the extension</LocalizedDescription>  
```
