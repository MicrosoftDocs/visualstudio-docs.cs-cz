# <a name="getting-started-with-source-control-vspackages"></a>Začínáme s balíčky VSPackage správy zdrojového kódu
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

VSPackage správy zdrojového kódu je pokročilejší alternativou k modulu plug-in správy zdrojového kódu. (Viz [vytváření modulu Plug-in zdrojového ovládacího prvku](../../extensibility/internals/creating-a-source-control-plug-in.md) pro další informace o zdrojové určují moduly plug-in.) VSPackage správy zdrojového kódu umožňuje úplnou kontrolu nad model správy zdrojového kódu, funkce a uživatelské rozhraní (UI) a integruje do [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] prostředí jako VSPackage.

 Popisuje možnosti pro řešení pro řízení zdrojů a poskytuje široké pokynů pro výběr cestu k odpovídající integrace.
