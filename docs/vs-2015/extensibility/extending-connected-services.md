# <a name="extending-connected-services"></a>Rozšíření připojené služby
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Aplikaci Visual Studio Connected Services SDK umožňuje poskytovatelé služeb vytvořit rozšíření sady Visual Studio, které vývojáři aplikací můžete snadno přidat do svého projektu bez opuštění integrovaného vývojového prostředí sady Visual Studio.
