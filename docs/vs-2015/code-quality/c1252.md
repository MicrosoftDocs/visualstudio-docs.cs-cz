# <a name="c1252"></a>C1252
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C1252: cyklická nebo chybějící závislost mezi moduly plug-in: vyžaduje GUID.  
  
 Nástroj Analýza kódu oznamuje toto upozornění v případě, že došlo k vnitřní chybě se závislostmi modulů plug-in, nikoli v analyzovaném kódu.
