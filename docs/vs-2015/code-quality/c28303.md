# <a name="c28303"></a>C28303
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28303: pro referenční parametr jazyka C++ <parameter_name> `_Deref_` byl v prvku nalezen dvojznačný operátor \<annotation> .  
  
 Toto upozornění se podobá upozornění C28302 a je hlášena, pokud `_Deref_` se v parametru používá další úroveň.  
  
 POZNÁMKOU SAL2 nevyžaduje použití další úrovně `_Deref_` při práci s parametry odkazu. Tato konkrétní anotace je nejednoznačná, protože úroveň odkázání odkazů je určena k tomu, aby bylo možné je opatřit poznámkami. Může být nutné použít `_At_` k odkazování na konkrétní objekt, který se má opatřit poznámkami.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění, protože použití `__deref_out_ecount(n)` je dvojznačné:  
  
```  
void ref(__deref_out_ecount(n) int **&buff, int &n)  
```  
  
 Výše uvedená poznámka mohla být interpretována jako:  
  
- odkaz na pole (z n) ukazatelů na celá čísla (interpretace SAL1)  
  
- odkaz na ukazatel na pole (z n) celých čísel (interpretace POZNÁMKOU SAL2)  
  
  Toto upozornění může vyřešit některá z následujících možností:  
  
```  
void ref(_Out_writes_(n) int **&buff, int &n)  
// or  
_At_(*buff), _Out_writes(n)) void ref(int **&buff, int &n)  
  
```
