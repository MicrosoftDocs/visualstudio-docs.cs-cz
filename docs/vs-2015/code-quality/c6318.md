# <a name="c6318"></a>C6318
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C6318: nesprávně definovaná __try/ \_ _except: použijte konstantu EXCEPTION_CONTINUE_SEARCH nebo jinou konstantu, která vyhodnocuje na nulu ve výrazu filtru výjimky. Kód v bloku obslužné rutiny výjimky není proveden.  
  
 Toto upozornění znamená, že pokud dojde k výjimce v chráněném bloku této strukturované obslužné rutiny výjimek, výjimka nebude zpracována, protože konstanta `EXCECPTION_CONTINUE_SEARCH` je použita ve výrazu filtru výjimky.  
  
 Tento kód je ekvivalentní chráněnému bloku bez bloku obslužných rutin výjimek, protože blok obslužné rutiny není spuštěn.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění:  
  
```  
#include <excpt.h>  
#include <stdio.h>  
  
void f (char *pch)  
{  
  __try   
  {  
    // assignment might fail  
    *pch = 0;  
  }   
  __except (EXCEPTION_CONTINUE_SEARCH) // warning 6318  
  {  
    puts("Exception Occurred");  
  }  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```  
#include <excpt.h>  
#include <stdio.h>  
#include <windows.h>  
  
void f (char *pch)  
{  
  __try   
  {  
    // assignment might fail  
    *pch = 0;  
  }   
  __except( (GetExceptionCode() == EXCEPTION_ACCESS_VIOLATION) ?  
                EXCEPTION_EXECUTE_HANDLER : EXCEPTION_CONTINUE_SEARCH )  
  {  
    puts("Access violation");  
  }  
}   
```
