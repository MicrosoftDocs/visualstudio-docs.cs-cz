# <a name="c28112"></a>C28112
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C28112: k proměnné, ke které se dá přistupovat přes propojenou funkci, se musí vždycky přistupovat prostřednictvím propojené funkce.  
  
 Podívejte se na *číslo*řádku: není vždy bezpečné přistupovat k proměnné, ke které se přistupuje prostřednictvím propojeného * rodiny funkcí jakýmkoli jiným způsobem.  
  
 Proměnnou, ke které se přistoupí pomocí rutiny podpory s podporou spolupracovníka, jako je například InterlockedCompareExchangeAcquire, je později k dispozici pomocí jiné metody. I když určité běžné přiřazení, přístup a porovnání k proměnným, které jsou používány propojenými * rutinami, lze bezpečně přistoupit pomocí jiné metody, riziko je dostatečně Skvělé, aby bylo možné prozkoumání jednotlivých instancí.  
  
## <a name="example"></a>Příklad  
 Následující příklad kódu vygeneruje toto upozornění:  
  
```  
inter_var --;  
...  
InterlockedIncrement(&inter_var);  
```  
  
 Následující příklad kódu předchází tomuto upozornění:  
  
```  
InterlockedDecrement(&inter_var);  
...  
InterlockedIncrement(&inter_var);  
```
