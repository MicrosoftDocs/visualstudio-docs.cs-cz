# <a name="c6388"></a>C6388
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6388: \<argument> možná není \<value> : to nedodržuje specifikace pro funkci \<function name> : řádky: x, y  
  
 Toto upozornění znamená, že v zadaném kontextu je použita Neočekávaná hodnota. Tato hodnota je obvykle hlášena pro hodnoty předané jako argumenty funkci, která to neočekává.  
  
## <a name="example"></a>Příklad  
 Následující kód jazyka C++ vygeneruje toto upozornění, protože DoSomething očekává hodnotu null, ale může být předána potenciálně jiná hodnota než null:  
  
```cpp  
  
#include <string.h>  
#include <malloc.h>  
#include <sal.h>  
  
void DoSomething( _Pre_ _Null_ void* pReserved );  
  
void f()  
{  
    void* p = malloc( 10 );  
    DoSomething( p );  // Warning C6388  
    // code...  
    free(p);  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující vzorový kód:  
  
```cpp  
  
#include <string.h>  
#include <malloc.h>  
#include <sal.h>  
  
void DoSomething( _Pre_ _Null_ void* pReserved );  
void f()  
{  
    void* p = malloc( 10 );  
    if (!p)  
    {  
        DoSomething( p );    
    }  
    else  
    {  
        // code...  
        free(p);  
    }  
}  
```  
  
 Všimněte si, že použití nástrah a Free má mnoho z pohledu nevracení paměti a výjimek. Chcete-li se vyhnout těmto typům nevracení a potížím s výjimkami, použijte mechanismy, které jsou k dispozici v knihovně C++ Standard Template Library (STL). Mezi ně patří [shared_ptr](https://msdn.microsoft.com/library/1469fc51-c658-43f1-886c-f4530dd84860), [unique_ptr](https://msdn.microsoft.com/library/acdf046b-831e-4a4a-83aa-6d4ee467db9a)a [Vector](https://msdn.microsoft.com/library/c1431ad8-c0b6-4dbb-89c4-5f651e432d7f). Další informace naleznete v tématu [inteligentní ukazatele](https://msdn.microsoft.com/library/909ef870-904c-49b6-b8cd-e9d0b7dc9435) a [Standardní knihovna C++](https://msdn.microsoft.com/library/a37d3ba3-58af-47c7-9ee2-441ccd7b77ee).
