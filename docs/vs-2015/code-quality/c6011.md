# <a name="c6011"></a>C6011
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6011: přesměrování ukazatele s hodnotou NULL \<name>  
  
 Toto upozornění signalizuje, že je zpětně indikován ukazatel s hodnotou null. Pokud je hodnota ukazatele neplatná, výsledek není definován.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění, protože volání metody \ může vracet hodnotu null, pokud není k dispozici dostatek paměti:  
  
```  
#include <malloc.h>  
  
void f( )  
{   
  char *p = ( char * ) malloc( 10 );  
  *p = '\0';  
  
  // code ...  
 free( p );  
}  
```  
  
 Chcete-li toto upozornění opravit, prověřte ukazatel na hodnotu null, jak je znázorněno v následujícím kódu:  
  
```  
#include <malloc.h>  
void f( )  
{  
  char *p = ( char * )malloc ( 10 );  
  if ( p )   
  {  
    *p = '\0';  
    // code ...  
  
    free( p );  
  }  
}  
```  
  
 Před zrušením odkazu na parametr je nutné přidělit paměť uvnitř funkce, jejíž parametry jsou opatřeny pomocí vlastnosti null v předběžné podmínce. Následující kód vygeneruje upozornění C6011, protože došlo k pokusu o zrušení odkazu na ukazatel s hodnotou null ( `pc` ) uvnitř funkce bez prvotního přidělení paměti:  
  
```  
#include <sal.h>  
using namespace vc_attributes;  
void f([Pre(Null=Yes)] char* pc)  
{  
  *pc='\0'; // warning C6011 - pc is null  
  // code ...  
}  
```  
  
 Použití typu \ a Free má mnoho nástrah z pohledu nevracení paměti a výjimek. Chcete-li se vyhnout těmto typům nevracení a potížím s výjimkami, použijte mechanismy, které jsou k dispozici v knihovně C++ Standard Template Library (STL). Mezi ně patří [shared_ptr](https://msdn.microsoft.com/library/1469fc51-c658-43f1-886c-f4530dd84860), [unique_ptr](https://msdn.microsoft.com/library/acdf046b-831e-4a4a-83aa-6d4ee467db9a)a [Vector](https://msdn.microsoft.com/library/c1431ad8-c0b6-4dbb-89c4-5f651e432d7f). Další informace naleznete v tématu [inteligentní ukazatele](https://msdn.microsoft.com/library/909ef870-904c-49b6-b8cd-e9d0b7dc9435) a [Standardní knihovna C++](https://msdn.microsoft.com/library/a37d3ba3-58af-47c7-9ee2-441ccd7b77ee).
