# <a name="ca0051"></a>CA0051

Nebyla vybrána žádná pravidla.

Tato chyba označuje, že je povolen nástroj pro analýzu spravovaného kódu, ale nebyly vybrány žádné pravidla. Chcete-li tuto chybu opravit, vyberte odpovídající pravidla.
