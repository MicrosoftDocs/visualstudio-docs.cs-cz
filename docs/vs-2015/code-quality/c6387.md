# <a name="c6387"></a>C6387
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Upozornění C6387: \<argument> může \<value> to být: Toto nedodržuje specifikace pro funkci \<function name> : řádky: x, y  
  
 Toto upozornění je vyvoláno, pokud je parametr funkce s poznámkou předán neočekávanou hodnotu. Například předání potenciálně hodnoty null parametru, který je označen pomocí `_In_` poznámky generuje toto upozornění.  
  
## <a name="example"></a>Příklad  
 Následující kód vygeneruje toto upozornění, protože parametr s hodnotou null je předán `f(char *)` :  
  
```cpp  
  
#include <sal.h>  
  
_Post_ _Null_ char * g();  
  
void f(_In_ char *pch);  
  
void main()  
{  
    char *pCh = g();  
    f(pCh); // Warning C6387  
}  
```  
  
 Chcete-li toto upozornění opravit, použijte následující kód:  
  
```cpp  
  
#include <sal.h>  
  
_Post_ _Notnull_ char * g();  
  
void f(_In_ char *pch);  
  
void main()  
{  
    char *pCh = g();  
    f(pCh);  
}  
```
