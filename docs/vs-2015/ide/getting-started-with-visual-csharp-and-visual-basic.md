# <a name="getting-started-with-visual-c-and-visual-basic"></a>Začínáme s jazykem Visual C# a Visual Basic
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ať už začínáte s jazykem Visual C#, nebo Microsoft Visual Basic, jste zde správně. Nápověda Začínáme je série postupných lekcí, které vás seznámí s jazyky Visual C# a Visual Basic. Bavte se a napište nějaký skvělý software.

 Další informace o vizuálu C++najdete v [tématu C++ vizuální prostředky](https://msdn.microsoft.com/vstudio/hh386302.aspx).

## <a name="in-this-section"></a>V tomto oddílu
 [Návod: Vytvoření jednoduché aplikace](../ide/walkthrough-create-a-simple-application-with-visual-csharp-or-visual-basic.md) Identifikujte základní okna v integrovaném vývojovém prostředí sady Visual Studio, optimalizujte vývojové prostředí tím, že nakonfigurujete nastavení a pak sestavíte jednoduchý program.

 [Kurz 1: vytvoření prohlížeče obrázků](../ide/tutorial-1-create-a-picture-viewer.md) Sestavte program, který načte obrázek ze souboru a zobrazí jej v okně. Naučte se přetáhnout prvky jako tlačítka a pole obrázků ve formuláři, nastavit jejich vlastnosti a plynule změnit velikost formuláře pomocí kontejnerů. Začněte psát kód.

 [Kurz 2: vytvoření časovaného matematického kvízu](../ide/tutorial-2-create-a-timed-math-quiz.md) Sestavení časované hry matematického kvízu, kde hráč musí odpovědět na čtyři náhodné aritmetické úlohy v určeném čase. Naučte se generovat náhodná čísla pomocí třídy `Random`, aktivovat události pomocí ovládacího prvku `Timer`, řídit tok programu pomocí příkazů `if else` a provádět základní aritmetické operace.

 [Kurz 3: vytvoření porovnávací hry](../ide/tutorial-3-create-a-matching-game.md) Vytvořte porovnávací hru, kde hráč musí hledat dvojice skrytých ikon. Naučte se ukládat objekty pomocí `List` objektu, použít smyčku `foreach`, udržovat přehled o stavu formuláře pomocí referenčních proměnných, vytvořit obslužnou rutinu události, kterou můžete použít s více objekty, a nechat časovač vyvolat právě jednou při spuštění.
