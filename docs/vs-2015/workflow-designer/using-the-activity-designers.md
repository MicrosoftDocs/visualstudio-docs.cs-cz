# <a name="using-the-activity-designers"></a>Používání návrhářů aktivit
@No__t_0 obsahuje řadu aktivit poskytovaných systémem, které můžete použít při vytváření aktivit a pracovních postupů. Tato část popisuje kategorie aktivit poskytovaných systémem a jejich návrhářů, které jsou k dispozici. Témata týkající se jednotlivých činností a jejich příslušných návrhářů v rámci každé z těchto kategorií popisují, co tyto aktivity dělají, a poskytují pokyny k jejich použití.

 [Tok řízení](../workflow-designer/control-flow-activity-designers.md) Návrháři aktivit sloužící k řízení toku v rámci pracovního postupu

 [Vývojový diagram](../workflow-designer/flowchart-activity-designers.md) Návrháři aktivit slouží k vytváření aktivit vývojového diagramu.

 [Stavový počítač](../workflow-designer/state-machine-activity-designers.md) Návrháři aktivit, kteří slouží k sestavování aktivit stavového počítače.

 [Zasílání zpráv](../workflow-designer/messaging-activity-designers.md) Návrháři aktivit používané ke správě aktivit zasílání zpráv.

 [Modul runtime](../workflow-designer/runtime-activity-designers.md) Návrháři aktivit sloužící k uchování a ukončení aktivit pracovních postupů.

 [Primitivní prvky](../workflow-designer/primitives-activity-designers.md) Návrháři aktivit používané ke správě primitivních aktivit, jako je přiřazení a zpoždění.

 [Transakce](../workflow-designer/transaction-activity-designers.md) Návrháři aktivit používané ke správě aktivit transakcí.

 [Kolekce](../workflow-designer/collection-activity-designers.md) Návrháři aktivit, kteří se používají ke správě aktivit shromažďování

 [Zpracování chyb](../workflow-designer/error-handling-activity-designers.md) Návrháři aktivit používané ke správě aktivit zpracování chyb.

 [Migrace](../workflow-designer/migration-activity-designers.md) Návrháři aktivit sloužící k vzájemné spolupráci mezi [!INCLUDE[vstecwinfx](../includes/vstecwinfx-md.md)] / [!INCLUDE[netfx35_short](../includes/netfx35-short-md.md)] a pracovními postupy [!INCLUDE[netfx40_short](../includes/netfx40-short-md.md)].
