# <a name="find-window-dialog-box"></a>Dialogové okno vyhledávacího okna
Použijte toto dialogové okno Vybrat a zobrazit její vlastnosti nebo zprávy. K zobrazení tohoto dialogového okna, zvolte **najít okno** z **Spy** nabídky.  
  
 K dispozici jsou následující nastavení:  
  
 **Nástroj pro vyhledávání**  
 Přetažení nástroj používaný k vyberte časové období.  
  
 **Skrýt nástroje Spy ++**  
 Odebere hlavní okno nástroje Spy ++ a zůstane **najít okno** dialogové okno zobrazené na jiné aplikace.  
  
 **Popisovač**  
 Jedinečné ID okna Najít.  
  
 **Popisek**  
 Text záhlaví okna.  
  
 **– Třída**  
 Třída okna.  
  
 **Styl**  
 Styl okna.  
  
 **Rect –**  
 Ohraničující obdélník pro okno.  
  
 **Zobrazit vlastnosti nebo zpráv**  
 Určuje, zda se zobrazí stisknutím OK [dialogové okno Vlastnosti okna](../debugger/window-properties-dialog-box.md) nebo otevřít [zobrazení zpráv](../debugger/messages-view.md).
