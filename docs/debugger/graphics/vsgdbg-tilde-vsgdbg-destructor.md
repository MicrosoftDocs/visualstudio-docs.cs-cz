# <a name="vsgdbgvsgdbg-destructor"></a>VsgDbg::~VsgDbg (destruktor)
Odstraní instanci `VsgDbg` třídy. Pokud se aktivně nahrává grafických informací, soubor protokolu grafika je dokončen a zavřít a uvolnění prostředků, které jste použili při aktivně zaznamenání grafických informací.  
  
## <a name="syntax"></a>Syntaxe  
  
```C++  
~VsgDbg();  
```
