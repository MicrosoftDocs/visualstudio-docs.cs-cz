# <a name="net-compiler-platform-package-version-reference"></a>Odkaz na verzi balíčku kompilátoru rozhraní .NET

V následující tabulce je uvedeno, které verze [balíčků platformy kompilátoru .NET (Roslyn)](https://www.nuget.org/packages/Microsoft.Net.Compilers/) jsou podporovány pro různé verze sady Visual Studio.

Jako příklad je třeba zajistit, aby váš vlastní analyzátor fungoval ve všech verzích sady Visual Studio 2017, měl by se zaměřit na verzi 2.0 microsoft.net.compilers.

| Verze balíčku Roslyn | Minimální podporovaná verze sady Visual Studio |
| - | - |
| 3.x | Visual Studio 2019 |
| 2.10.0 | Visual Studio 2017 verze 15.9 |
| 2.9.0 | Visual Studio 2017 verze 15.8 |
| 2.8.2 | Visual Studio 2017 verze 15.7 |
| 2.7.0 | Visual Studio 2017 verze 15.6 |
| 2.6.1 | Visual Studio 2017 verze 15.5 |
| 2.4.0 | Visual Studio 2017 verze 15.4 |
| 2.3.2 | Visual Studio 2017 verze 15.3 |
| 2.2.0 | Visual Studio 2017 verze 15.2 |
| 2.1.0 | Visual Studio 2017 verze 15.1 |
| 2.0.0 | Visual Studio 2017 RTM |
| 1.3.2 | Aktualizace Visual Studia 2015 3 |
| 1.2.2 | Aktualizace Visual Studia 2015 2 |
| 1.1.1 | Aktualizace Visual Studia 2015 1 |
| 1.0.1 | Visual Studio 2015 RTM |

> [!TIP]
> Pro balíčky Roslyn, kde minimální podporovaná verze sady Visual Studio je verze sady Visual Studio 2017, jsou podporovány také všechny verze Sady Visual Studio 2019, protože byly později.
