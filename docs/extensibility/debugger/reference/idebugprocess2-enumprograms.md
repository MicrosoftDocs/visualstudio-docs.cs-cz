# <a name="idebugprocess2enumprograms"></a>IDebugProcess2::EnumPrograms
Načte seznam všech programů obsažených v tomto procesu.

## <a name="syntax"></a>Syntaxe

```cpp
HRESULT EnumPrograms( 
   IEnumDebugPrograms2** ppEnum
);
```

```csharp
int EnumPrograms( 
   out IEnumDebugPrograms2 ppEnum
);
```

## <a name="parameters"></a>Parametry
`ppEnum`\
mimo Vrátí objekt [IEnumDebugPrograms2](../../../extensibility/debugger/reference/ienumdebugprograms2.md) , který obsahuje seznam všech programů, které jsou v procesu.

## <a name="return-value"></a>Návratová hodnota
 V případě úspěchu vrátí. `S_OK` jinak vrátí kód chyby.
