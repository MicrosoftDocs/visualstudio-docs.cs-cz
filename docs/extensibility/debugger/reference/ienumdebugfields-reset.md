# <a name="ienumdebugfieldsreset"></a>IEnumDebugFields::Reset
Tato metoda obnoví výčtu první prvek.  
  
## <a name="syntax"></a>Syntaxe  
  
```cpp  
HRESULT Reset(void);  
```  
  
```csharp  
int Reset();  
```  
  
#### <a name="parameters"></a>Parametry  
 Žádné
