# <a name="module_info_flags"></a>MODULE_INFO_FLAGS
Určuje stav symbolů pro modul.

## <a name="syntax"></a>Syntax

```cpp
enum enum_MODULE_INFO_FLAGS {
   MIF_SYMBOLS_LOADED = 0x0001
};
typedef DWORD MODULE_INFO_FLAGS;
```

```csharp
public enum enum_MODULE_INFO_FLAGS {
   MIF_SYMBOLS_LOADED = 0x0001
};
```

## <a name="fields"></a>Pole
 `MIF_SYMBOLS_LOADED`\
 V modulu byla načtena alespoň jedna sada symbolů (jinak nebyly načteny žádné symboly).

## <a name="remarks"></a>Poznámky
 Tato hodnota je vrácena metodou [getsymbolsearchinfo –](../../../extensibility/debugger/reference/idebugsymbolsearchevent2-getsymbolsearchinfo.md) .
