# <a name="list-of-available-services"></a>Seznam dostupných služeb

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] a sada Visual Studio SDK podporují následující služby. Některé balíčky nabízejí své vlastní služby, které tu nejsou uvedené – například jazykové služby nemají jeden identifikátor GUID služby. K vyhledání identifikátoru GUID služby jazyka v registru musíte použít název jazyka.

K získání primárního rozhraní nebo rozhraní zobrazených s každou službou použijte identifikátory GUID služeb uvedené tady nebo získané z nějakého jiného zdroje (například jazykových služeb).
