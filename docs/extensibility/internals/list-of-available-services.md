# <a name="list-of-available-services"></a>Seznam dostupných služeb

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] a sada Visual Studio SDK podporuje následující služby. Některé balíčky nabízejí vlastní služby, které tady nejsou uvedené, například jazykové služby nemají jeden identifikátor GUID služby. K vyhledání identifikátoru GUID služby jazyka v registru je nutné použít název jazyka.

Použijte zde uvedené identifikátory GUID služby nebo získané z nějakého jiného zdroje (například jazykové služby), abyste získali primární rozhraní nebo rozhraní zobrazená u jednotlivých služeb.
