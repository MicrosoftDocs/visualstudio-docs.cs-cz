# <a name="could-not-retrieve-schema-information-for-database-object-object-name"></a>Nelze načíst informace o schématu pro objekt databáze \<název objektu >

Tato zpráva se obvykle zobrazují při aktualizaci objektu v **Průzkumníka serveru** nebo **Průzkumník databáze** zkopírován do schránky, odstraněna z databáze a pak vložit do návrháře. Vzhledem k tomu, že databázový objekt již existuje, zobrazí se tato zpráva.

Chcete-li opravit tuto chybu, vyberte **OK** zavřete dialogové okno.
