# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Upozornění: Změny byly provedeny na dialogovém okně Konfigurace chování, které nebyly použity

Upozornění: Změny byly provedeny na dialogovém okně Konfigurace chování, které nebyly použity. Chcete-li aplikovat provedené změny?

**Konfigurace chování** dialogové okno umožňuje konfigurovat `Insert`, `Update`, a `Delete` chování pro všechny třídy, které jsou k dispozici. Tato zpráva se zobrazí, když vyberete nový **třídy** a **chování** kombinace a předchozí změnu nebyl dosud nebyly použity.

## <a name="to-apply-the-change-and-continue"></a>Použít změny a pokračovat

- Klikněte na tlačítko **Ano** na použití změny pro vybrané **třídy** a **chování**.

## <a name="to-cancel-the-previous-change-and-continue"></a>Předchozí změnu zrušit a pokračovat

- Klikněte na tlačítko **ne** pro zrušení změn pro vybrané **třídy** a **chování**.
