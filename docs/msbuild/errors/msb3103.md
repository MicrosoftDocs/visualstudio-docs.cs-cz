# <a name="msb3103-invalid-resx-file"></a>MSB3103: Neplatný soubor Resx

K této chybě dochází v případě problému se souborem prostředků .resx. Když se zobrazí tato chyba, zobrazí se další chyba s více podrobnostmi. Další informace najdete v druhé chybě.
