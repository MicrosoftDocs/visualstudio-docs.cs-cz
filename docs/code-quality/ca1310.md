# <a name="ca1310-specify-stringcomparison-for-correctness"></a>CA1310: Zadejte StringComparison pro správnost

|Položka|Hodnota|
|-|-|
|RuleId|CA1310|
|Kategorie|Microsoft. Globalization|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina
Operace porovnání řetězců používá přetížení metody, které nenastavuje <xref:System.StringComparison> parametr a používá ve výchozím nastavení porovnávání řetězců specifické pro jazykovou verzi. Proto se chování bude lišit v závislosti na nastavení národního prostředí aktuálního uživatele.

## <a name="rule-description"></a>Popis pravidla
Metoda porovnání řetězců, která používá porovnání řetězců specifických pro jazykovou verzi, může mít potenciálně nechtěné běhové chování, které se neshoduje s záměrem uživatele. Doporučuje se použít přetížení s <xref:System.StringComparison> parametrem pro správnost a přehlednost záměru.

Toto pravidlo označuje metody porovnání řetězců, které ve výchozím nastavení používají hodnotu specifickou pro jazykovou verzi <xref:System.StringComparison> . Další informace naleznete v tématu [porovnání řetězců, které používají aktuální jazykovou verzi](/dotnet/standard/base-types/best-practices-strings#string-comparisons-that-use-the-current-culture).

> [!NOTE]
> Pokud chcete zobrazit porušení pro všechny metody porovnání řetězců bez ohledu na výchozí porovnání řetězců používané metodou, použijte [CA1307: zadejte StringComparison pro přehlednost](ca1307.md) místo toho.

## <a name="how-to-fix-violations"></a>Jak opravit porušení
Chcete-li opravit porušení tohoto pravidla, změňte metody porovnání řetězců na přetížení, které přijímají <xref:System.StringComparison> výčet jako parametr. Například změňte `String.Compare(str1, str2)` na `String.Compare(str1, str2, StringComparison.Ordinal)` .

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění
Je bezpečné potlačit upozornění od tohoto pravidla, pokud knihovna nebo aplikace není určena k lokalizaci.
