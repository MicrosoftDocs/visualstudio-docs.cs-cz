# <a name="extended-correctness-rules-rule-set-for-managed-code"></a>Sada pravidel Rozšířená pravidla správnosti pro spravovaný kód
Sada pravidel pravidla správnosti rozšířené Microsoft maximalizuje logiku a framework využití chyby, které jsou hlášeny nástroje Analýza kódu. Navíc je důraz na konkrétní scénáře, jako je interoperabilita modelů COM a mobilní aplikace. Měli byste zvážit, včetně toto pravidlo nastavit, pokud jeden z těchto scénářů vztahuje na projektu nebo k nalezení dalších problémů ve vašem projektu.

 Sada pravidel pravidla správnosti rozšířené Microsoft obsahuje pravidla, které jsou v pravidle základní pravidla správnosti Microsoft nastavit. Základní pravidla správnosti obsahují pravidla, které jsou v pravidle Microsoft Minimální doporučená pravidla nastavit. Další informace najdete v části [sada pravidel základní pravidla správnosti pro spravovaný kód](../code-quality/basic-correctness-rules-rule-set-for-managed-code.md) a [sada pravidel spravovaná doporučená pravidla pro spravovaný kód](../code-quality/managed-recommended-rules-rule-set-for-managed-code.md)
