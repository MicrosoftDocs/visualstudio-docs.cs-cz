# <a name="ca5371-use-xmlreader-for-schema-read"></a>CA5371: Použít XmlReader pro čtení schématu

|Položka|Hodnota|
|-|-|
|CheckId|CA5371|
|Kategorie|Microsoft.Security|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Zpracování nedůvěryhodného vstupu XML s <xref:System.Xml.Schema.XmlSchema.Read%2A?displayProperty=nameWithType> vytvořením instance bez `XmlReader` objektu může potenciálně vést k odepření služby, zpřístupnění informací a útokům na straně serveru pro falšování. Tyto útoky jsou povoleny nedůvěryhodnou verzí DTD a zpracování schématu XML, což umožňuje zahrnutí bomby XML a škodlivých externích entit do XML. Pouze pomocí `XmlReader` je možné zakázat DTD. Zpracování vloženého schématu XML `XmlReader` má ve `ProhibitDtd` `ProcessInlineSchema` výchozím nastavení vlastnost a nastavenou na hodnotu false v .NET Framework počínaje verzí 4,0. Další možnosti, jako například `Stream` , `TextReader` , a `XmlSerializationReader` nelze zakázat zpracování DTD.

## <a name="rule-description"></a>Popis pravidla

Zpracování nedůvěryhodných schémat DTD a XML může povolit načítání nebezpečných externích odkazů. Použití pomocí `XmlReader` zabezpečeného překladače nebo pomocí vloženého zpracování schématu DTD a XML je zakázané. Toto pravidlo detekuje kód, který používá <xref:System.Xml.Schema.XmlSchema.Read%2A?displayProperty=nameWithType> metodu bez použití `XmlReader` parametru.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Použijte `XmlSchema.Read(XmlReader, *)` přetížení.

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Toto upozornění můžete potenciálně potlačit, pokud <xref:System.Xml.Schema.XmlSchema.Read%2A?displayProperty=nameWithType> je metoda vždy používána ke zpracování XML pocházející z důvěryhodného zdroje, a proto nemůže být poškozena.

## <a name="pseudo-code-examples"></a>Příklady kódu pseudo

### <a name="violation"></a>Selhání

Následující ukázka pseudo kódu znázorňuje vzor zjištěný tímto pravidlem.
Typ prvního parametru <xref:System.Xml.Schema.XmlSchema.Read%2A?displayProperty=nameWithType> není `XmlReader` .

```csharp
using System.IO;
using System.Xml.Schema;
...
public void TestMethod(Stream stream, ValidationEventHandler validationEventHandler)
{
    XmlSchema.Read(stream, validationEventHandler);
}
```

### <a name="solution"></a>Řešení

```csharp
using System.IO;
using System.Xml.Schema;
...
public void TestMethod(XmlReader reader, ValidationEventHandler validationEventHandler)
{
    XmlSchema.Read(reader, validationEventHandler);
}
```
