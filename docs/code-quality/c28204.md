# <a name="c28204"></a>C28204
Warning C28204: \<funkce >: pro\<paramName > je nutné zadat pouze jedno z těchto přetížení a druhý v \<název_souboru filename > (\<line >): obojí nebo ani jeden z nich musí být opatřen poznámkami.

 Toto upozornění se oznamuje v případě, že poznámky obsahují chybu.
