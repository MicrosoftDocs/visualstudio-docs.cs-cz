# <a name="ca1831-use-asspan-instead-of-range-based-indexers-for-string-when-appropriate"></a>CA1831: Tam, kde je to možné, používat u řetězců místo indexerů založených na rozsahu metodu AsSpan

|Položka|Hodnota|
|-|-|
|CheckId|CA1831|
|Kategorie|Microsoft. Performance|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

V řetězci je použit indexer rozsahu a hodnota je implicitně přiřazena `ReadOnlySpan<char>` .

## <a name="rule-description"></a>Popis pravidla

Toto pravidlo je aktivována, když použijete rozsah-indexer na řetězec a přiřadíte ho k typu rozsahu. Indexer rozsahu pro <xref:System.Span%601> je operace, která není kopírováním <xref:System.Span%601.Slice%2A?#System_Span_1_Slice_System_Int32_System_Int32_> , ale pro indexer rozsahu na řetězci se <xref:System.String.Substring%2A?#System_String_Substring_System_Int32_System_Int32_> použije metoda místo <xref:System.Span%601.Slice%2A?#System_Span_1_Slice_System_Int32_System_Int32_> . Tím se vytvoří kopie požadované části řetězce. Tato kopie je obvykle zbytečná, pokud se implicitně používá jako <xref:System.ReadOnlySpan%601> hodnota nebo <xref:System.ReadOnlyMemory%601> . Pokud kopie není zamýšlená, použijte <xref:System.MemoryExtensions.AsSpan%2A?#System_MemoryExtensions_AsSpan_System_String_AsSpan> metodu, aby nedocházelo k zbytečné kopii. Pokud je kopie určena, buď ji přiřaďte k místní proměnné, nebo přidejte explicitní přetypování. Analyzátor generuje pouze v případě, že je pro výsledek operace indexeru rozsahu použit implicitní přetypování.

### <a name="detects"></a>Zjistit

Implicitní převod:

`ReadOnlySpan<char> slice = str[a..b];`

### <a name="does-not-detect"></a>Nedetekuje

Explicitní převod:

`ReadOnlySpan<char> slice = (ReadOnlySpan<char>)str[a..b];`

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Chcete-li opravit porušení tohoto pravidla, použijte <xref:System.MemoryExtensions.AsSpan%2A?#System_MemoryExtensions_AsSpan_System_String_AsSpan> místo <xref:System.Range> indexeru založeného na řetězci řetězec, abyste se vyhnuli vytváření zbytečných kopií dat.

```csharp
public void TestMethod(string str)
{
    // The violation occurs
    ReadOnlySpan<char> slice = str[1..3];
    ...
}
```

```csharp
public void TestMethod(string str)
{
    // The violation fixed with AsSpan extension method
    ReadOnlySpan<char> slice = str.AsSpan()[1..3];
    ...
}
```

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Pokud je vytvoření kopie zamýšlené, je bezpečné potlačit porušení tohoto pravidla. Kromě [obvyklých způsobů, jak potlačit upozornění analýzy kódu](in-source-suppression-overview.md), můžete také přidat explicitní přetypování, aby se zabránilo upozornění.

```csharp
public void TestMethod(string str)
{
    // The violation occurs.
    ReadOnlySpan<char> slice = str[1..3];
    ...
}
```

```csharp
public void TestMethod(string str)
{
    // The violation avoided with explicit casting.
    ReadOnlySpan<char> slice = (ReadOnlySpan<char>)str[1..3];
    ...
}
```
