# <a name="c6383"></a>C6383
Upozornění C6383: přetečení vyrovnávací paměti kvůli převodu počtu prvků na počet bajtů: u parametru \<number > při volání \<function > je očekáván počet prvků.

 Toto upozornění označuje, že počet bajtů, které nejsou konstantní, se předává, když je vyžadován počet prvků. Obvykle k tomu dochází, když je proměnná vynásobena `sizeof` typu, ale analýza kódu naznačuje, že je vyžadován počet prvků.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění:

```
#include <string.h>

void f( wchar_t* t, wchar_t* s, int n )
{
  // code...
  wcsncpy (t, s, n*sizeof(wchar_t)); // warning 6383
  // code ...
}
```

 Chcete-li toto upozornění opravit, vynásobte proměnnou hodnotou `sizeof` typu, jak je znázorněno v následujícím kódu:

```
void f( wchar_t* t, wchar_t* s, int n )
{
  // code
  wcsncpy (t, s, n);
  // code ...
}
```

 Následující kód opravuje toto upozornění pomocí funkce bezpečného zpracování řetězců:

```
void f(wchar_t* t, wchar_t* s, size_t n)
{
  // code...
  wcsncpy_s( t, sizeof(s), s, n );
  // code...
}
```
