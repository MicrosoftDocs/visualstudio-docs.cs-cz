# <a name="ca0053"></a>CA0053
Sestavení pravidla nelze načíst.

To znamená, že sestavení pravidla nelze načíst pomocí reflexe. K tomu může dojít v případě, že uživatel má vlastní pravidla, která nebyla správně sestavena, nebo nástroj pro analýzu nemůže načíst sestavení z jiných důvodů, například přístup k systému souborů byl odepřen.
