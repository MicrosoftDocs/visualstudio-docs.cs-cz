# <a name="ca0056"></a>CA0056
Soubor projektu má nesprávnou verzi nástroje pro analýzu.

K tomuto upozornění dochází, pokud je FxCopCmd.exe vyvolána ručně pomocí souboru projektu, který má nesprávnou verzi nástroje pro analýzu kódu.
