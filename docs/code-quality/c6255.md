# <a name="c6255"></a>C6255
Warning C6255: _alloca značí selhání zvýšením výjimky přetečení zásobníku. Zvažte místo toho použití _malloca

 Toto upozornění signalizuje, že volání `_alloca` bylo zjištěno mimo místní zpracování výjimky. `_alloca` by mělo být vždy voláno z chráněného rozsahu obslužné rutiny výjimky, protože může vyvolat výjimku přetečení zásobníku při selhání. Pokud je to možné, místo použití `_alloca` zvažte použití `_malloca`, což je bezpečnější verze `_alloca`.

## <a name="example"></a>Příklad
 Následující kód vygeneruje toto upozornění, protože `_alloca` může generovat výjimku:

```
#include <windows.h>

void f( )
{
  void *p = _alloca(10);
  // code ...
}
```

 Chcete-li toto upozornění opravit, použijte `_malloca` a přidejte obslužnou rutinu výjimky, jak je znázorněno v následujícím kódu:

```
#include <windows.h>
#include <malloc.h>

void f( )
{
  void *p;
  int errcode;
  __try
  {
    p = _malloca(10);
    // code...
    _freea(p);
  }
  __except( (GetExceptionCode() == STATUS_STACK_OVERFLOW ) ?
              EXCEPTION_EXECUTE_HANDLER : EXCEPTION_CONTINUE_SEARCH )
  {
    errcode = _resetstkoflw();
    // code ...
  }
}
```
