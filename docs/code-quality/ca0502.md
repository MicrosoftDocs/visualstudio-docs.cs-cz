# <a name="ca0502"></a>CA0502
Nepodporovaný jazyk

K této chybě dochází, když se uživatel pokusí analyzovat projekt ASP.NET, který obsahuje jednu nebo více stránek nebo tříd napsaných v jiném jazyce než C# nebo Visual Basic. Chcete-li tuto chybu opravit, odeberte soubor z projektu nebo zakažte analýzu kódu.
