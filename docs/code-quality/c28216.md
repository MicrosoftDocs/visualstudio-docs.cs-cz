# <a name="c28216"></a>C28216
Upozornění C28216: \_check\_vrácení poznámky\_ platí pouze pro parametr post-Conditions pro konkrétní parametr funkce.

 Anotace `_Check_return_` byla použita v jiném kontextu než `__post`; může být potřeba, aby byl modifikátor `__post` (nebo `__drv_out`) nebo mohl být nesprávně umístěn.
