# <a name="ca2326-do-not-use-typenamehandling-values-other-than-none"></a>CA2326: Nepoužívejte jiné hodnoty TypeNameHandling než None

|Položka|Hodnota|
|-|-|
|CheckId|CA2326|
|Kategorie|Microsoft.Security|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Toto pravidlo je vyvoláno, pokud je splněna jedna z následujících podmínek:
- [Newtonsoft.Js. ](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_TypeNameHandling.htm)Je odkazována hodnota výčtu TypeNameHandling, která je jiná než `None` .
- Celočíselná hodnota představující nenulovou hodnotu je přiřazena proměnné [TypeNameHandling](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_TypeNameHandling.htm) .

## <a name="rule-description"></a>Popis pravidla

[!INCLUDE[insecure-deserializers-description](includes/insecure-deserializers-description-md.md)]

Toto pravidlo vyhledá [Newtonsoft.Js. TypeNameHandling](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_TypeNameHandling.htm) hodnoty jiné než `None` . Pokud chcete deserializovat pouze v případě, že je [Newtonsoft.Js. Serializace. ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm) je určena k omezení deserializovaných typů, zakažte toto pravidlo a místo toho povolte pravidla [CA2327](ca2327.md), [CA2328](ca2328.md), [CA2329](ca2329.md)a [CA2330](ca2330.md) .

## <a name="how-to-fix-violations"></a>Jak opravit porušení

- Pokud [TypeNameHandling](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_TypeNameHandling.htm)je `None` to možné, použijte hodnotu TypeNameHandling.
- Proveďte serializovanou manipulaci s daty. Po serializaci kryptograficky podepisují Serializovaná data. Před deserializací ověřte kryptografický podpis. Chránit kryptografický klíč před zveřejněním a návrhem pro střídání klíčů.
- Omezí deserializovatelné typy. Implementace vlastního [Newtonsoft.Jsv. Serializace. ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm). Před deserializací pomocí Json.NET zajistěte, aby vlastní [ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm) byla určena ve vlastnosti [Newtonsoft.Json.JsonSerializerSettings. SerializationBinder](https://www.newtonsoft.com/json/help/html/P_Newtonsoft_Json_JsonSerializerSettings_SerializationBinder.htm) . V přepsaném [Newtonsoft.Jsna. Serialization. ISerializationBinder. BindToType](https://www.newtonsoft.com/json/help/html/M_Newtonsoft_Json_Serialization_ISerializationBinder_BindToType.htm) metoda, pokud je typ neočekávaný, vraťte `null` nebo vyvolejte výjimku pro zastavení deserializace.
  - Pokud omezíte deserializované typy, možná budete chtít zakázat toto pravidlo a povolit pravidla [CA2327](ca2327.md), [CA2328](ca2328.md), [CA2329](ca2329.md)a [CA2330](ca2330.md). Pravidla [CA2327](ca2327.md), [CA2328](ca2328.md), [CA2329](ca2329.md)a [CA2330](ca2330.md) vám zajistí, že použijete [ISerializationBinder](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_Serialization_ISerializationBinder.htm) při použití hodnot [TypeNameHandling](https://www.newtonsoft.com/json/help/html/T_Newtonsoft_Json_TypeNameHandling.htm) jiných než `None` .

## <a name="pseudo-code-examples"></a>Příklady kódu pseudo

### <a name="violation"></a>Selhání

```csharp
using Newtonsoft.Json;

public class ExampleClass
{
    public JsonSerializerSettings Settings { get; }

    public ExampleClass()
    {
        Settings = new JsonSerializerSettings();
        Settings.TypeNameHandling = TypeNameHandling.All;    // CA2326 violation.
    }
}
```

```vb
Imports Newtonsoft.Json

Public Class ExampleClass
    Public ReadOnly Property Settings() As JsonSerializerSettings

    Public Sub New()
        Settings = New JsonSerializerSettings()
        Settings.TypeNameHandling = TypeNameHandling.All    ' CA2326 violation.
    End Sub
End Class
```

### <a name="solution"></a>Řešení

```csharp
using Newtonsoft.Json;

public class ExampleClass
{
    public JsonSerializerSettings Settings { get; }

    public ExampleClass()
    {
        Settings = new JsonSerializerSettings();

        // The default value of Settings.TypeNameHandling is TypeNameHandling.None.
    }
}
```

```vb
Imports Newtonsoft.Json

Public Class ExampleClass
    Public ReadOnly Property Settings() As JsonSerializerSettings

    Public Sub New()
        Settings = New JsonSerializerSettings()

        ' The default value of Settings.TypeNameHandling is TypeNameHandling.None.
    End Sub
End Class
```
