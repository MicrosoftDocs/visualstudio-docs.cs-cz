# <a name="c28214"></a>C28214
upozornění C28214: atribut názvy parametrů musí být p1... p9

 Toto upozornění označuje, že při vytváření deklarace poznámky názvy parametrů jsou omezeny na p1... p9. Toto upozornění nikdy provedeno v běžném provozu.
