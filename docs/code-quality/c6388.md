# <a name="c6388"></a>C6388
Upozornění C6388: \<argument > nesmí být \<value >: to nedodržuje specifikace pro funkci \<function >: Řádky: x, y

 Toto upozornění znamená, že v zadaném kontextu je použita Neočekávaná hodnota. Tato hodnota je obvykle hlášena pro hodnoty předané jako argumenty funkci, která to neočekává.

## <a name="example"></a>Příklad
 Následující C++ kód vygeneruje toto upozornění, protože DoSomething očekává hodnotu null, ale může být předána potenciálně jiná hodnota než null:

```cpp

#include <string.h>
#include <malloc.h>
#include <sal.h>

void DoSomething( _Pre_ _Null_ void* pReserved );

void f()
{
    void* p = malloc( 10 );
    DoSomething( p );  // Warning C6388
    // code...
    free(p);
}
```

 Chcete-li toto upozornění opravit, použijte následující vzorový kód:

```cpp

#include <string.h>
#include <malloc.h>
#include <sal.h>

void DoSomething( _Pre_ _Null_ void* pReserved );
void f()
{
    void* p = malloc( 10 );
    if (!p)
    {
        DoSomething( p );
    }
    else
    {
        // code...
        free(p);
    }
}
```

 Všimněte si, že použití nástrah a Free má mnoho z pohledu nevracení paměti a výjimek. Chcete-li se vyhnout těmto typům nevrácených chyb a potížím s výjimkami, použijte C++ mechanismy, které jsou k dispozici pomocí knihovny Standard Template Library (STL). Patří mezi ně [shared_ptr](/cpp/standard-library/shared-ptr-class), [unique_ptr](/cpp/standard-library/unique-ptr-class)a [Vector](/cpp/standard-library/vector). Další informace najdete v tématu [inteligentní ukazatele](/cpp/cpp/smart-pointers-modern-cpp) a [ C++ standardní knihovna](/cpp/standard-library/cpp-standard-library-reference).
