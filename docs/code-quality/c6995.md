# <a name="c6995"></a>C6995
upozornění C6995: nepovedlo se uložit soubor protokolu XML

 Toto upozornění značí, že nástroj pro analýzu kódu nelze vytvářet vadou protokol, který je výstup analýzy kódu.

 Tato chyba může znamenat chybu disku nebo znamenat, že nemáte oprávnění k vytvoření souboru v určeném adresáři.
