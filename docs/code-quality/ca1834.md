# <a name="ca1834-use-stringbuilderappendchar-for-single-character-strings"></a>CA1834: Pro jednoznakové řetězce používat metodu StringBuilder.Append(char)

|Položka|Hodnota|
|-|-|
|RuleId|CA1834|
|Kategorie|Microsoft. Performance|
|Zásadní změna|Nenarušující|

## <a name="cause"></a>Příčina

Toto pravidlo je vyvoláno, když je do metody předán řetězec délky jednotky <xref:System.Text.StringBuilder.Append%2A> .

## <a name="rule-description"></a>Popis pravidla

Při volání `StringBuilder.Append` s řetězcem délky jednotky zvažte použití `const char` spíše než délky jednotky, `const string` aby se zlepšil výkon.

## <a name="how-to-fix-violations"></a>Jak opravit porušení

Porušení lze buď opravit ručně, nebo v některých případech pomocí rychlých akcí pro opravu kódu v aplikaci Visual Studio. Příklady:

##### <a name="example-1"></a>Příklad 1
Vyvolání `StringBuilder.Append` s řetězcovým literálem délky jednotky:
```csharp
using System; 
using System.Text;
 
namespace TestNamespace 
{ 
    class TestClass 
    { 
        private void TestMethod() 
        { 
            StringBuilder sb = new StringBuilder();
            sb.Append("a");
        } 
    } 
}
```
> [!TIP]
> Pro toto pravidlo v aplikaci Visual Studio je k dispozici oprava kódu. Pokud ho chcete použít, umístěte kurzor na porušení a stiskněte **CTRL** + **.** (tečka). **Pokud je to možné, vyberte možnost při použití použít StringBuilder. Append (Char).** ze seznamu zobrazených možností.
>
> ![Oprava kódu pro CA1834 – použití StringBuilder. Append (Char) pro řetězce s jedním znakem](media/ca1834-codefix.png)

##### <a name="fix-applied-by-visual-studio"></a>Oprava použitá v aplikaci Visual Studio
```csharp
using System; 
using System.Text;
 
namespace TestNamespace 
{ 
    class TestClass 
    { 
        private void TestMethod() 
        { 
            StringBuilder sb = new StringBuilder();
            sb.Append('a');
        } 
    } 
}
```

V některých případech, například při použití pole třídy délky jednotky `const string` , oprava kódu není navrhována aplikací Visual Studio (ale analyzátor stále je aktivován). Tyto instance vyžadují ruční opravu.

##### <a name="example-2"></a>Příklad 2
Vyvolání `StringBuilder.Append` s `const string` polem třídy délky jednotky:
```cs
using System;
using System.Text;

namespace TestNamespace
{
    public class Program
    {
        public const string unitString = "a";

        static void Main(string[] args)
        {
            StringBuilder sb = new StringBuilder();
            sb.Append(unitString);
        }
    }
}
```
Po pečlivé analýze se `unitString` dá změnit na, `char` aniž by to způsobilo chyby sestavení. 

##### <a name="fix"></a>Oprava
```cs
using System;
using System.Text;

namespace TestNamespace
{
    public class Program
    {
        public const char unitString = 'a';

        static void Main(string[] args)
        {
            StringBuilder sb = new StringBuilder();
            sb.Append(unitString);
        }
    }
}
```

## <a name="when-to-suppress-warnings"></a>Kdy potlačit upozornění

Pokud se nerozhodnete zvýšit výkon při použití nástroje, je bezpečné potlačit porušení tohoto pravidla `StringBuilder` .
