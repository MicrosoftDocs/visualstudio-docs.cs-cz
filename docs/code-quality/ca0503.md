# <a name="ca0503"></a>CA0503

Vlastnost je zastaralá. Použít vlastnost převádějící

CA0503 upozornění je generována, když vlastnost analýza zastaralého kódu obsahuje hodnotu v konfiguračním souboru sestavení nebo projektu. Chcete-li vyřešit toto upozornění, použijte zadanou vlastnost, která ji nahrazuje.
