# <a name="ca0055"></a>CA0055
Nepovedlo se načíst soubor:\<cesta >

 Nástroj pro analýzu kódu nebylo možné načíst cílový soubor, který byl zadán pro analýzu. K tomu dochází, pokud soubor není nalezen, nebo byl odepřen přístup k souboru. Obvykle k tomu dojde, pokud uživatel se rozhodli přepsat vlastnost vstupní sestavení v sadě Visual Studio nebo vyvolat fxcopcmd.exe ručně.
