# <a name="how-to-add-comments-to-class-diagrams"></a>Postupy: Přidání komentářů do diagramů tříd

Můžete použít obrazce komentářů k přidávání poznámek k diagramům tříd. Obrazec komentáře obsahuje jednu vlastnost, **text**, do které můžete zadat text. Tvary komentářů existují pouze na ploše diagramu a nejsou v kódu.

Komentář se nachází v zobrazení diagramu tříd v **Návrhář tříd**. Otevřete-li druhý diagram třídy na stejný projekt, komentáře vytvořené v prvním zobrazení nejsou viditelné. Pokud diagram odstraníte, odstraní se také všechny komentáře, které obsahovaly.

Můžete změnit velikost obrazce komentáře, ale nemůžete měnit jiné aspekty jeho vzhledu, jako je jeho barva pozadí, písmo nebo velikost písma.

## <a name="to-add-a-comment"></a>Přidání komentáře

1. Přetáhněte komentář z **panelu nástrojů návrhář tříd** do diagramu tříd.

2. Klikněte na nový obrazec komentáře v diagramu a zadejte požadovaný text.
