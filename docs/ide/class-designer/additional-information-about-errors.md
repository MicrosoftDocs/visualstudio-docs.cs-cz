# <a name="class-designer-errors"></a>Chyby Návrhář – třída

**Třídy návrháře** nesleduje umístění zdrojových souborů, takže změnou strukturu projektu nebo přesunutím zdrojové soubory v projektu může způsobit **návrhář tříd** ke ztrátě informací o typu, například je běžné Změňte typ zdroje typedef, základní třídy a typy přidružení. Může dojít chybě, jako **třída Návrhář nemůže zobrazit tento typ**. Opravte případné chyby, přetažením upraveném nebo přemístěné zdrojový kód na diagramu tříd znovu, můžete ho zobrazit.

## <a name="resources"></a>Prostředky

Pomoc s další chyby a upozornění můžete najít v následujících zdrojích informací:

- [Práce s Visual C++ – kód](working-with-visual-cpp-code.md) obsahuje pokyny k odstraňování informace o zobrazení C++ v diagramu tříd.
- [Fórum pro Visual Studio návrhář tříd](http://go.microsoft.com/fwlink/?LinkId=160754) poskytuje fórum pro otázky o **návrhář tříd**.
