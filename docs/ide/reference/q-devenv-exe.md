# <a name="-devenvexe"></a>/? (devenv.exe)

Zobrazí okno se zprávou se seznamem všech `devenv` přepínačů s krátkým popisem každého přepínače.

## <a name="syntax"></a>Syntax

```shell
devenv /?
```
