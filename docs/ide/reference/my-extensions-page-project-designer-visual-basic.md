# <a name="my-extensions-page-project-designer-visual-basic"></a>Stránka Moje rozšíření, návrhář projektu (Visual Basic)
Použití **Moje rozšíření** stránky **Návrhář projektu** ke správě `My` rozšíření oboru názvů ve vašem projektu. `My` rozšíření oboru názvů umožňují přizpůsobit `My` obor názvů přidání vlastní členů. Informace o vytváření vlastních `My` rozšíření oboru názvů, najdete v části [rozšíření My Namespace v jazyce Visual Basic](/dotnet/visual-basic/developing-apps/customizing-extending-my/extending-the-my-namespace).

 Pro přístup k **Moje rozšíření** klikněte dvakrát **Můj projekt** pro váš projekt uzel v **Průzkumníku řešení**. Když **Návrhář projektu** se zobrazí, klikněte na tlačítko **Moje rozšíření** kartě.

## <a name="uielement-list"></a>Seznam prvků uživatelského rozhraní
 Následující možnosti vám umožňují přidat nebo odebrat `My` rozšíření oboru názvů ve vašem projektu. A `My` rozšíření oboru názvů musí být nejprve nainstalována jako Visual Studio položky šablonu, která má být k dispozici, který se má přidat. Informace o publikování a instalaci `My` rozšíření oboru názvů, najdete v části [balení a nasazení Moje rozšíření vlastní](/dotnet/visual-basic/developing-apps/customizing-extending-my/packaging-and-deploying-custom-my-extensions).

 **Rozšíření oboru názvů My**

 Tento seznam obsahuje všechny `My` rozšíření oboru názvů, které jsou nainstalované v projektu.

 **Přidání rozšíření**

 Kliknutím na toto tlačítko Přidat je nainstalován `My` rozšíření oboru názvů do projektu. Seznam všech možných `My` rozšíření oboru názvů se zobrazí. Vyberte `My` rozšíření oboru názvů, který chcete přidat do projektu a klikněte na **OK** ho přidejte.

 **Odebrat rozšíření**

 Vyberte jeden nebo více odkazů v **rozšíření oboru názvů My** seznamu a potom na toto tlačítko k odebrání `My` rozšíření oboru názvů z projektu.
