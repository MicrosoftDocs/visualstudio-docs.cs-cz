# <a name="visible-timeline-profile"></a>Profil viditelné časové osy
Profil viditelné časové osy pro zobrazení vláken blokování poskytuje statistické informace a odkazy na sestavy. Jako přiblížení, oddálení, posuňte se vodorovně, kanály skrytí nebo zobrazení kanály, jsou čísla ve aktivní Legenda změnit tak, aby odrážela, co je aktuálně v zobrazení. Chcete-li zobrazit sestavu o položce v legendě, klikněte na položku.
