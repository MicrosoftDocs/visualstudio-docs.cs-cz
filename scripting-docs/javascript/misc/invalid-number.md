# <a name="invalid-number"></a>Číslo je neplatné
Neplatné číslo (ErrbadNumber 1013)  
  
 Pokusili jste se použít číslo, které není platné v JavaScriptu.  
  
 Informace o platných číslech naleznete v tématu [Number Object](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Global_Objects/Number).
