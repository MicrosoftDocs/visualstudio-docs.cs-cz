# <a name="the-use-of-a-keyword-for-an-identifier-is-invalid"></a>Použití klíčového slova pro identifikátor je neplatné
1048: použití klíčového slova pro identifikátor je neplatné  
  
 Klíčové slovo nelze použít jako název proměnné, funkce nebo další identifikátor definovaný uživatelem.  
  
 Další informace o klíčových slovech najdete v tématu [vyhrazená slova jazyka JavaScript](../../javascript/reference/javascript-reserved-words.md) a [vyhrazená slova jazyka JavaScript budoucí](../../javascript/reference/javascript-future-reserved-words.md).
