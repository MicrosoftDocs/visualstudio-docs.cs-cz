# <a name="expected-var"></a>Byl očekáván příkaz 'var'
Byl očekáván příkaz var (ERRnoVar 1034).  
  
 V případě potřeby se nepodařilo přidat klíčové slovo var.  
  
 Další informace o klíčovém slově var naleznete v tématu [var Statement](../../javascript/reference/var-statement-javascript.md).
