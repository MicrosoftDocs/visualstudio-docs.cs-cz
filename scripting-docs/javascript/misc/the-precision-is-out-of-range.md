# <a name="the-precision-is-out-of-range"></a>Přesnost je mimo rozsah
Pokoušíte se o předat neplatný argument funkce **Number.prototype.toPrecision**. Argument **toPrecision** musí 1 až 21 (včetně).  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Ujistěte se, že argument `toPrecision` není příliš velký či příliš malý.
