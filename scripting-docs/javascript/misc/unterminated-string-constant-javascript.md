# <a name="unterminated-string-constant-javascript"></a>Neukončená řetězcová konstanta (JavaScript)
Vaše řetězcová konstanta s uzavírací uvozovky neukončil. Řetězcové konstanty musí být uzavřena mezi dva znaky uvozovek.  
  
> [!NOTE]
>  Můžete použít odpovídající dvojic jednoduchých nebo dvojitých uvozovek. Dvojité uvozovky, může být obsažený v rámci řetězce v jednoduchých uvozovkách a apostrofech může být obsažený v rámci řetězce v uvozovkách.  
  
### <a name="to-correct-this-error"></a>Oprava této chyby  
  
-   Přidejte uzavírací uvozovky na konci řetězce.
