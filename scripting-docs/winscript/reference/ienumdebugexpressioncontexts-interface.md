# <a name="ienumdebugexpressioncontexts-interface"></a>IEnumDebugExpressionContexts – rozhraní
Vytvoří výčet kolekce `IDebugExpressionContexts` objekty.  
  
 Kromě metod zděděno z `IUnknown`, `IEnumDebugExpressionContexts` rozhraní poskytuje následující metody.  
  
## <a name="methods-in-vtable-order"></a>Metody v tabulce Vtable pořadí  
  
|Metoda|Popis|  
|------------|-----------------|  
|[IEnumDebugExpressionContexts::Next](../../winscript/reference/ienumdebugexpressioncontexts-next.md)|Načte zadaný počet segmentů v pořadí výčtu.|  
|[IEnumDebugExpressionContexts::Skip](../../winscript/reference/ienumdebugexpressioncontexts-skip.md)|Přeskočí zadaný počet segmentů v posloupnosti výčtu.|  
|[IEnumDebugExpressionContexts::Reset](../../winscript/reference/ienumdebugexpressioncontexts-reset.md)|Návrat na začátek v sekvenci výčtu.|  
|[IEnumDebugExpressionContexts::Clone](../../winscript/reference/ienumdebugexpressioncontexts-clone.md)|Vytvoří enumerátor, který obsahuje stejného stavu jako aktuální enumerátor.|
