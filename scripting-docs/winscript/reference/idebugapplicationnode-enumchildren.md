# <a name="idebugapplicationnodeenumchildren"></a>IDebugApplicationNode::EnumChildren
Vytvoří výčet podřízené uzly uzlu této aplikace.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT EnumChildren(  
   IEnumDebugApplicationNodes**  pperddp  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pperddp`  
 [out] Výčet tento uzel podřízené uzly.  
  
## <a name="return-value"></a>Návratová hodnota  
 Vrátí metodu `HRESULT`. Možné hodnoty zahrnují hodnoty v následující tabulce, ale nejsou na ně omezeny.  
  
|Hodnota|Popis|  
|-----------|-----------------|  
|`S_OK`|Metoda byla úspěšná.|  
  
## <a name="remarks"></a>Poznámky  
 Tato metoda vytvoří výčet podřízené uzly uzlu této aplikace.
