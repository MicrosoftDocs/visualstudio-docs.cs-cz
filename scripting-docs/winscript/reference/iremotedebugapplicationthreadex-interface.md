# <a name="iremotedebugapplicationthreadex-interface"></a>IRemoteDebugApplicationThreadEx – rozhraní
Představuje vlákno při provádění v rámci konkrétní aplikace.  
  
 Kromě metod zděděno z `IUnknown`, `IRemoteDebugApplicationThreadEx` rozhraní poskytuje následující metody.  
  
## <a name="methods-in-vtable-order"></a>Metody v tabulce Vtable pořadí  
  
|Metoda|Popis|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|Vytvoří výčet kontexty globálním výrazu pro všechny jazyky, které jsou spuštěné v této přístup z více vláken.|
