# <a name="idebugapplicationgetbreakflags"></a>IDebugApplication::GetBreakFlags
Vrátí aktuální rozdělení příznaky pro aplikaci.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT GetBreakFlags(  
   APPBREAKFLAGS*                   pabf,  
   IRemoteDebugApplicationThread**  pprdatSteppingThread  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pabf`  
 [out] Aktuální rozdělení příznaky pro aplikaci.  
  
 `pprdatSteppingThread`  
 [out] Aktuálně spuštěných vláken.  
  
## <a name="return-value"></a>Návratová hodnota  
 Vrátí metodu `HRESULT`. Možné hodnoty zahrnují hodnoty v následující tabulce, ale nejsou na ně omezeny.  
  
|Hodnota|Popis|  
|-----------|-----------------|  
|`S_OK`|Metoda byla úspěšná.|  
  
## <a name="remarks"></a>Poznámky  
 Tato metoda vrátí aktuální rozdělení příznaky pro aplikaci.
