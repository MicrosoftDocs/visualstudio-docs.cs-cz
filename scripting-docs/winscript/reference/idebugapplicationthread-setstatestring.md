# <a name="idebugapplicationthreadsetstatestring"></a>IDebugApplicationThread::SetStateString
Nastaví popis stavu přístup z více vláken.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
HRESULT SetStateString(  
   LPCOLESTR  pstrState  
);  
```  
  
#### <a name="parameters"></a>Parametry  
 `pstrState`  
 [v] Popis stavu přístup z více vláken.  
  
## <a name="return-value"></a>Návratová hodnota  
 Vrátí metodu `HRESULT`. Možné hodnoty zahrnují hodnoty v následující tabulce, ale nejsou na ně omezeny.  
  
|Hodnota|Popis|  
|-----------|-----------------|  
|`S_OK`|Metoda byla úspěšná.|  
  
## <a name="remarks"></a>Poznámky  
 Tato metoda nastaví popis stavu přístup z více vláken.
